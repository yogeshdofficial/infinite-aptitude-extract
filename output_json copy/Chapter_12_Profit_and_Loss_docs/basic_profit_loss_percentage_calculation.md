# Pattern: Basic Profit/Loss Percentage Calculation

## 1. Pattern Overview
This pattern provides the foundational methodology for calculating the percentage of gain or loss relative to the initial investment (Cost Price). It is essential for evaluating the financial performance of any transaction involving the buying and selling of goods.

## 2. Key Formulas
The following formulas are the building blocks for all basic profit and loss calculations:

*   **Profit (Gain):** $$\text{Gain} = \text{S.P.} - \text{C.P.}$$
*   **Loss:** $$\text{Loss} = \text{C.P.} - \text{S.P.}$$
*   **Profit Percentage:** $$\text{Profit \%} = \left( \frac{\text{Gain}}{\text{C.P.}} \times 100 \right)$$
*   **Loss Percentage:** $$\text{Loss \%} = \left( \frac{\text{Loss}}{\text{C.P.}} \times 100 \right)$$
*   **S.P. from Profit %:** $$\text{S.P.} = \text{C.P.} \times \left( \frac{100 + \text{Gain \%}}{100} \right)$$
*   **S.P. from Loss %:** $$\text{S.P.} = \text{C.P.} \times \left( \frac{100 - \text{Loss \%}}{100} \right)$$

## 3. When to Use This Pattern
Use this pattern when a problem provides the purchase price and the sale price and asks for the relative efficiency of the trade in percentage terms.

*   **Keywords:** "Percent profit," "Percent loss," "Gain percent," "Sold at a profit of," "Incurred a loss of."
*   **Given:** Cost Price (C.P.) and Selling Price (S.P.).
*   **To Find:** The percentage increase or decrease relative to the original cost.

## 4. Core Concept & Theory
The fundamental principle of this pattern is that **profit or loss is always reckoned on the Cost Price (C.P.)**. Because the C.P. represents the initial capital invested, it serves as the "base" (denominator) for all percentage calculations. If the Selling Price is higher than the Cost Price, the difference is a gain; if lower, it is a loss.

## 5. Step-by-Step Solution Method
1.  **Identify C.P. and S.P.:** Extract the purchase price and selling price from the problem statement.
2.  **Determine the outcome:** Compare S.P. and C.P. If $S.P. > C.P.$, calculate $\text{Gain} = S.P. - C.P.$. If $C.P. > S.P.$, calculate $\text{Loss} = C.P. - S.P.$.
3.  **Apply the Percentage Formula:** Divide the Gain or Loss by the **C.P.** (never the S.P.).
4.  **Convert to Percentage:** Multiply the resulting fraction by 100 to get the final percentage value.

## 6. Worked Examples

**Example 1: Basic Profit Calculation**
*Given:* Mansi purchased a car for ₹2,50,000 and sold it for ₹3,48,000.
*Solution:*
1. C.P. = 2,50,000; S.P. = 3,48,000.
2. Since S.P. > C.P., Profit = 3,48,000 - 2,50,000 = 98,000.
3. Profit % = $(98,000 / 2,50,000) \times 100$.
4. Result: $0.392 \times 100 = 39.2\%$.

**Example 2: Basic Loss Calculation**
*Given:* C.P. = ₹2516 and S.P. = ₹2272.
*Solution:*
1. C.P. = 2516; S.P. = 2272.
2. Since C.P. > S.P., Loss = 2516 - 2272 = 244.
3. Loss % = $(244 / 2516) \times 100$.
4. Result: $\approx 9.7\%$.

## 7. Recognition Clues & Keywords
*   **"Reckoned on C.P.":** This is the golden rule. If you see a problem asking for profit percentage, immediately look for the C.P.
*   **"Sold at a gain of x%":** This implies $S.P. = (100+x)\%$ of $C.P.$.
*   **Avoid Confusion:** Do not calculate profit percentage based on the Selling Price unless explicitly stated (which is rare in standard arithmetic).

## 8. Common Mistakes
1.  **Using S.P. as the denominator:** Students often divide the profit by the Selling Price. This is incorrect; always use the Cost Price.
2.  **Mixing up Gain and Loss:** Always check if $S.P. > C.P.$ or $C.P. > S.P.$ before choosing the formula.
3.  **Arithmetic Errors with Zeros:** When dealing with large numbers (like lakhs), ensure the number of zeros in the numerator and denominator match before canceling.
4.  **Forgetting the 100 multiplier:** The result of the division is a decimal; forgetting to multiply by 100 will result in an incorrect percentage value.

## 9. Practice Tips
*   **Mental Math:** Practice converting simple fractions to percentages (e.g., $1/4 = 25\%$, $1/5 = 20\%$) to speed up calculations.
*   **Related Patterns:** Once comfortable, study "False Weight Calculations" (Patterns XIII-XV) and "Successive Percentage Changes," as these build directly upon these basic formulas.
*   **Verification:** Always check if your answer makes logical sense. If you bought for 100 and sold for 120, your profit must be 20%. If you get a different number, re-check your division.
