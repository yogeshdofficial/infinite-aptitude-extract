# Problems On Numbers — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Entity:** Determine if the question involves a single number, two numbers, a fraction, or digits of a number.
2. **Translate to Math:** Convert words to variables (e.g., "two-digit number" $\to 10x+y$; "consecutive" $\to x, x+1$).
3. **Select Pattern:** Match the structure to the Quick-Recognition Table below.
4. **Formulate:** Write the equation(s) based on the "Given" information.
5. **Solve:** Use algebraic manipulation (substitution, elimination, or quadratic factoring).
6. **Verify:** Check if the result satisfies all conditions (e.g., positive integer, digit range 0-9).

## Formula Bank

**[Algebraic Identities]**
$$x \times (\frac{a}{b}x) = P$$
→ *produces: quadratic equation for $x$*
→ *use when: number multiplied by its fraction*

$$x^2 - bx - c = 0$$
→ *produces: roots of the number*
→ *use when: solving complex algebraic relations*

**[Consecutive Numbers]**
$$\text{Sum} = n \times \text{middle term}$$
→ *produces: the central value*
→ *use when: sum of odd count consecutive numbers*

$$\text{Average} = \frac{\text{Sum}}{\text{Count}}$$
→ *produces: middle value for even count*
→ *use when: finding consecutive sequence bounds*

**[Digit Problems]**
$$\text{Number} = 10x + y$$
→ *produces: two-digit number value*
→ *use when: digits are manipulated*

$$\text{Number} = 100h + 10t + u$$
→ *produces: three-digit number value*
→ *use when: hundreds place is involved*

**[Fraction Problems]**
$$\text{Fraction} = \frac{x}{y}$$
→ *produces: the unknown fraction*
→ *use when: numerator/denominator changes given*

$$a^2 = \text{Product} \times \text{Quotient}$$
→ *produces: square of greater fraction*
→ *use when: product and quotient provided*

**[Linear Equations]**
$$\text{Number} = \frac{\text{Sum of limits}}{2}$$
→ *produces: equidistant number*
→ *use when: "as much greater as less"*

$$|x-y| = \sqrt{(x+y)^2 - 4xy}$$
→ *produces: difference of numbers*
→ *use when: sum and product known*

**[Ratio and Proportion]**
$$\text{Part} = \text{Total} \times \frac{\text{Ratio}}{\text{Sum of Ratios}}$$
→ *produces: individual number value*
→ *use when: total sum is given*

$$(ax)^2 + (bx)^2 + (cx)^2 = S$$
→ *produces: multiplier $x$*
→ *use when: sum of squares given*

**[Reciprocal Relations]**
$$\frac{1}{x} + \frac{1}{y} = \frac{x+y}{xy}$$
→ *produces: relation between sum and product*
→ *use when: sum of reciprocals given*

## Step Sequences

**Algebraic Identities:** Define $x$ → Set equation → Solve quadratic → Verify.
**Consecutive Numbers:** Find average → Determine middle → Expand sequence → Check.
**Digit Problems:** Define $10x+y$ → Set digit equations → Solve system → Reconstruct.
**Fraction Problems:** Define $\frac{x}{y}$ → Apply changes → Solve system → Simplify.
**Linear Equations:** Assign variables → Translate conditions → Solve system → Validate.
**Ratio and Proportion:** Assign $kx$ → Sum/Product equation → Solve $x$ → Multiply.
**Reciprocal Relations:** Set $x + \frac{1}{x}$ → Form quadratic → Solve roots → Check.

---

## Fastest Tricks

**Linear Equations:** Equidistant numbers are always the average of the two limits.
**Ratio:** For sum of squares, divide total by sum of squared ratio parts.
**Reciprocal:** If $x + \frac{1}{x} = \frac{a}{b}$, check if $\frac{a}{b} = \frac{x}{y} + \frac{y}{x}$.
**Digit Problems:** If digits interchange, the difference is always $9 \times (\text{difference of digits})$.

---

## Trap Watch

**Algebraic Identities:** Forgetting to square $x$ → Write $x^2$.
**Consecutive Numbers:** Assuming average is a number → Check parity.
**Digit Problems:** Writing $x+y$ for number → Use $10x+y$.
**Fraction Problems:** Mixing up numerator/denominator → Label clearly.
**Ratio:** Squaring ratio before multiplying by $x$ → Square $x$ too.
**Reciprocal:** Forgetting negative roots → Check both signs.

---

## Last-Minute Reminders

A two-digit number is $10x+y$, never $x+y$.
The average of an odd number of consecutive integers is always the middle term.
When the product and quotient of two fractions are known, their product is the square of the larger fraction.
Always verify if the calculated number fits the constraints of the problem (e.g., digits must be 0-9).
If a quadratic equation has two roots, check if the question specifies "natural numbers" or "positive integers."
