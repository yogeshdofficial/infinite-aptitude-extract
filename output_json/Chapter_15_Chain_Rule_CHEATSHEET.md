# Chain Rule — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Variables:** List all entities (Men, Days, Hours, Work, etc.).
2. **Classify Relationship:** Determine if variables are Direct (as one increases, other increases) or Indirect (as one increases, other decreases).
3. **Assign Groups:** Label variables as Group 1 (known) and Group 2 (unknown).
4. **Select Formula:** Use the Chain Rule formula $\frac{M_1 D_1 H_1}{W_1} = \frac{M_2 D_2 H_2}{W_2}$ for complex problems, or simple ratios for basic ones.
5. **Standardize Units:** Ensure all units match (e.g., convert weeks to days, grams to kg).
6. **Solve & Sanity Check:** Calculate $x$; check if the result is logically consistent with the change in input variables.

## Formula Bank

**[Compound Proportion]**
$$\frac{M_1 \times D_1 \times H_1}{W_1} = \frac{M_2 \times D_2 \times H_2}{W_2}$$
→ *produces: missing variable in complex scenarios*
→ *use when: 3+ variables change simultaneously*

**[Direct Proportion]**
$$\frac{A_1}{B_1} = \frac{A_2}{B_2}$$
→ *produces: unknown quantity at constant rate*
→ *use when: variables change in same direction*

**[Indirect Proportion]**
$$M_1 \times D_1 = M_2 \times D_2$$
→ *produces: unknown time or worker count*
→ *use when: total work is constant*

**[Work Equivalence]**
$$\text{Total Work} = \text{Efficiency} \times \text{Time}$$
→ *produces: total capacity of a group*
→ *use when: solving "or/and" worker problems*

## Step Sequences

**Compound Proportion:** Identify variables → Assign groups → Set up ratio → Solve for $x$.
**Direct Proportion:** Find unit rate → Multiply by quantity → Answer.
**Indirect Proportion:** Multiply known pair → Divide by known variable → Answer.
**Work Equivalence:** Convert to common unit → Calculate total work → Solve for unknown → Answer.

---

## Fastest Tricks

* **Compound:** Use $x = \text{Initial} \times (\text{Ratio}_1) \times (\text{Ratio}_2) \dots$ based on direct/indirect logic.
* **Direct:** Use cross-multiplication: $A_1 \times B_2 = A_2 \times B_1$.
* **Indirect:** If workers decrease by factor $k$, time increases by factor $k$.
* **Work Equivalence:** Convert all workers to one type (e.g., all women) before calculating.

---

## Trap Watch

* **Compound:** Mixing direct/indirect ratios → Check logic per variable.
* **Direct:** Forgetting to convert units (e.g., weeks to days) → Convert all first.
* **Indirect:** Assuming direct proportion → Check if work is constant.
* **Work Equivalence:** Adding men/women directly → Convert to common unit first.

---

## Last-Minute Reminders

Work is always placed in the denominator of the chain rule formula.
Always verify if the "work" is the same or if it has changed (e.g., $\frac{2}{5}$ vs $\frac{3}{5}$).
When "additional men" are asked for, subtract the original count from your final result.
In indirect proportion, if the number of workers decreases, the time taken must increase.
Always check if the question asks for "total days" or "additional days" required.
