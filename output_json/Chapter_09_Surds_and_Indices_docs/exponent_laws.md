# Exponent Laws

## Overview
These questions involve simplifying complex expressions containing powers, roots, or decimals. The central idea is to express all numbers in the equation as powers of the same base so that exponents can be combined or equated.

---

## Recognition Clues
* **Visual:** Presence of numbers with exponents, roots (e.g., $\sqrt[3]{x}$), or decimals (e.g., $0.064$).
* **Given:** Multiple terms with different bases (e.g., $9, 8, 72$) or decimal bases.
* **Asked:** Simplify the expression or find the value of an unknown exponent ($x$ or $?$).
* **Keywords:** "Simplify," "Evaluate," "Find the value of," or an equation with a variable in the exponent.

---

## Key Formulas

### Power of a Power
$$
(a^m)^n = a^{m \times n}
$$
**Variables:** $a$ = base, $m, n$ = exponents.
**When to use:** When a term with an exponent is raised to another power.
**Worked example:** $(27)^{\frac{2}{3}} = (3^3)^{\frac{2}{3}} = 3^{3 \times \frac{2}{3}} = 3^2 = 9$.

### Product of Same Bases
$$
a^m \times a^n = a^{m+n}
$$
**Variables:** $a$ = common base, $m, n$ = exponents.
**When to use:** When multiplying terms that share the same base.
**Worked example:** $16^{0.32} \times 16^{0.18} = 16^{0.32+0.18} = 16^{0.5} = 4$.

### Negative Exponents
$$
a^{-n} = \frac{1}{a^n} \quad \text{and} \quad \left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n
$$
**Variables:** $a, b$ = base, $n$ = exponent.
**When to use:** When the exponent is negative; flip the base to make it positive.
**Worked example:** $(\frac{8}{125})^{-\frac{4}{3}} = (\frac{125}{8})^{\frac{4}{3}} = (\frac{5}{2})^4 = \frac{625}{16}$.

## Solution Framework

1. **Standardize Bases:** Convert all numbers into prime bases (e.g., $2, 3, 5$) or a common base found in the equation.
2. **Convert Decimals:** Change decimals to fractions (e.g., $0.064 = \frac{64}{1000} = (\frac{4}{10})^3$) to reveal the base.
3. **Apply Power Rule:** Use $(a^m)^n = a^{mn}$ to simplify nested exponents.
4. **Combine Terms:** Use $a^m \times a^n = a^{m+n}$ to group terms with identical bases.
5. **Equate Exponents:** If $a^x = a^y$, then $x = y$. Solve the resulting linear equation for the unknown.

---

## Shortcut Tricks

* **Trick:** If you see $9^x \times 8^x$, immediately write it as $(9 \times 8)^x = 72^x$.
* **Why it works:** The rule $a^n \times b^n = (ab)^n$ allows combining different bases if the exponents are identical.
* **When to use:** When the bases are different but the exponents are the same.
* **Example:** $9^{12.5} \times 8^{12.5} = 72^{12.5}$.

---

## Common Mistakes

* **Mistake:** Multiplying the base by the exponent (e.g., $3^3$ as $3 \times 3 = 9$ instead of $27$).
* **Why it happens:** Confusing the base with the exponent.
* **Fix:** Always write the base as a prime power first.
* **Mistake:** Forgetting to flip the fraction when the exponent is negative.
* **Why it happens:** Rushing the sign change.
* **Fix:** Apply the reciprocal rule $(\frac{a}{b})^{-n} = (\frac{b}{a})^n$ before calculating.

## Worked Example (Step-by-Step)

**Question:** Solve: $(0.064) \times (0.4)^7 = (0.4)^? \times (0.0256)^2$

**Solution:**
1. **Standardize:** Recognize $0.064 = (0.4)^3$ and $0.0256 = (0.4)^4$.
2. **Substitute:** $(0.4)^3 \times (0.4)^7 = (0.4)^x \times ((0.4)^4)^2$.
3. **Apply Power Rule:** $(0.4)^3 \times (0.4)^7 = (0.4)^x \times (0.4)^8$.
4. **Combine:** $(0.4)^{3+7} = (0.4)^{x+8} \Rightarrow (0.4)^{10} = (0.4)^{x+8}$.
5. **Equate:** $10 = x + 8 \Rightarrow x = 2$.

**Answer:** $2$

## Similar Patterns
**Logarithms:** Distinguished by the presence of "log" notation; exponents are solved using log properties rather than base-matching.

## Revision Summary
* **Key formula:** $a^m \times a^n = a^{m+n}$ and $(a^m)^n = a^{mn}$.
* **Spot it by:** Expressions with powers, roots, or decimals.
* **First move:** Convert all numbers to the same prime base.
* **Fastest method:** Combine bases using $a^n \times b^n = (ab)^n$ if exponents match.
* **Biggest trap:** Forgetting to flip the fraction for negative exponents.
