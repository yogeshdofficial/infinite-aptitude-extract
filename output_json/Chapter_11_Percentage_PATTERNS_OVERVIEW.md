# Percentage — Chapter Overview

## What This Chapter Is About

Percentage is the study of proportions expressed as parts of a whole, where the "whole" is always standardized to 100. It serves as the universal language for comparing quantities of different sizes, allowing us to analyze changes, growth, and distributions across various fields like finance, demographics, and commerce. Every problem in this chapter revolves around the relationship between a base value, a part of that value, and the resulting rate per hundred.

---

## Core Concepts

**[The Base of 100]** — Percentage is simply a fraction where the denominator is fixed at 100. When you see $x\%$, you are looking at $\frac{x}{100}$; this standardization allows you to compare a 5% increase on a small salary directly against a 5% increase on a large investment.

**[The Reference Value (The "Of" Value)]** — In any percentage problem, identifying the "base" or the "original" value is critical. If a question asks "What percent of $A$ is $B$?", $A$ is your base (the denominator), and $B$ is the part you are measuring.

**[Percentage as a Multiplier]** — Percentages act as scaling factors. If a quantity increases by 20%, you are essentially multiplying the original value by $1.20$ (or $1 + \frac{20}{100}$); understanding this decimal conversion allows you to calculate complex changes in a single step rather than through multiple additions.

**[Relative Change vs. Absolute Change]** — Absolute change is the raw difference (e.g., "the price went up by ₹50"), while percentage change is the relative impact (e.g., "the price went up by 10%"). Competitive exams almost always test your ability to convert absolute differences into these relative percentage terms.

---

## Key Terms Glossary

**Percentage** — A rate or proportion expressed as a fraction of 100. Denoted by the symbol $\%$.

**Fraction** — A numerical representation of a part of a whole, expressed as $\frac{numerator}{denominator}$. Converting percentages to fractions (e.g., $25\% = \frac{1}{4}$) is a primary tool for simplifying calculations.

**Base Value** — The original quantity or the "whole" from which a percentage is calculated. In the phrase "10% of 500," 500 is the base value.

**Depreciation** — A reduction in the value of an asset over time, usually expressed as a percentage decrease per year.

**Successive Change** — A scenario where a percentage change is applied to a value, and then a subsequent percentage change is applied to the *new* resulting value, rather than the original base.

## Core Formulas

### Percentage Change (Increase/Decrease)
$$
\text{New Value} = \text{Original Value} \times \left(1 \pm \frac{R}{100}\right)
$$
**Variables:** $R$ = rate of percentage change.
**When to use:** When a quantity grows or shrinks by a fixed percentage.
**Key insight:** Use $+$ for increase and $-$ for decrease.
**Worked example:** A ₹200 item increases by 10%: $200 \times (1 + 0.10) = 220$.

### Consumption Adjustment
$$
\text{Reduction} = \left( \frac{R}{100 + R} \right) \times 100\%
$$
**Variables:** $R$ = percentage increase in price.
**When to use:** When price increases but total expenditure must remain constant.
**Key insight:** The denominator increases because the price is higher.
**Worked example:** If price rises 25%, reduction = $\frac{25}{125} \times 100 = 20\%$.

### Population/Depreciation (Compound)
$$
\text{Final Value} = P \left( 1 \pm \frac{R}{100} \right)^n
$$
**Variables:** $P$ = initial value, $R$ = rate, $n$ = number of time periods.
**When to use:** When a value changes by the same percentage repeatedly over time.
**Key insight:** This is identical to the compound interest formula.
**Worked example:** 1000 population grows 10% for 2 years: $1000 \times (1.1)^2 = 1210$.

## Chapter-Wide Traps

**Confusing the Base:** Using the "new" value as the denominator when the question asks for a percentage of the "original" value → Always re-read the question to identify which number represents the "whole."

**Successive Change Error:** Adding percentages directly (e.g., 10% increase + 10% increase = 20% increase) → Always calculate the first change, then apply the second change to the *new* result.

**Unit Mismatch:** Calculating percentages between different units (e.g., grams vs. kilograms) → Always convert all quantities to the same unit before performing the percentage calculation.
