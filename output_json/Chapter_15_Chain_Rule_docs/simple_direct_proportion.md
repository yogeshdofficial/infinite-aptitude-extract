# 1. Pattern Overview
Simple Direct Proportion is a mathematical relationship where two quantities change in the same direction—if one increases, the other increases proportionally, and if one decreases, the other decreases proportionally. This pattern is essential for solving real-world problems involving scaling, unit pricing, and resource allocation.

# 2. Key Formulas
For two quantities $A$ and $B$ that are directly proportional, the relationship is expressed as:

**The Proportion Ratio:**
$$\frac{A_1}{B_1} = \frac{A_2}{B_2}$$

**The Cross-Multiplication Formula:**
$$A_1 \times B_2 = A_2 \times B_1$$

**Solving for the Unknown ($x$):**
$$x = \frac{A_2 \times B_1}{A_1}$$

# 3. When to Use This Pattern
Use this pattern when a problem describes two variables that move in tandem.
*   **Keywords:** "More... more...", "Less... less...", "At the same rate", "Proportional to".
*   **Recognition Clues:** Look for a scenario where you have a known pair (e.g., 5 apples cost $2) and you need to find a new value based on a change in one of those variables (e.g., how much do 12 apples cost?).
*   **Given:** Two initial values ($A_1, B_1$) and one new value ($A_2$).
*   **To Find:** The corresponding unknown value ($x$ or $B_2$).

# 4. Core Concept & Theory
The fundamental idea is **constancy of ratio**. If two items are directly proportional, the ratio between them remains constant regardless of the quantity. By setting up an equation where the ratio of the first set equals the ratio of the second set, we create a balanced equation that allows us to solve for the missing piece using basic algebra.

# 5. Step-by-Step Solution Method
1.  **Identify the Variables:** Determine the two quantities being compared (e.g., Wheat vs. Days).
2.  **Determine the Relationship:** Ask: "If one increases, does the other increase?" If yes, it is **Direct Proportion**.
3.  **Set up the Proportion:** Write the relationship as a ratio: $A_1 : A_2 :: B_1 : x$.
4.  **Apply Cross-Multiplication:** Multiply the extremes and the means ($A_1 \cdot x = A_2 \cdot B_1$).
5.  **Solve for $x$:** Isolate $x$ by dividing the product of the known diagonal by the remaining value.

# 6. Worked Examples

**Example 1: Cost of Goods**
*Given:* A canteen requires 105 kg of wheat for 7 days. How many kg are required for 58 days?
*Solution:*
1. Identify: Wheat ($W$) and Days ($D$).
2. Relationship: More days = More wheat (Direct).
3. Setup: $7 : 58 :: 105 : x$
4. Calculate: $7 \times x = 58 \times 105$
5. Solve: $x = \frac{58 \times 105}{7} = 58 \times 15 = 870$.
*Result:* 870 kg of wheat.

**Example 2: Work Output**
*Given:* If 5 machines produce 200 units in an hour, how many units will 8 machines produce in the same time?
*Solution:*
1. Identify: Machines ($M$) and Units ($U$).
2. Relationship: More machines = More units (Direct).
3. Setup: $5 : 8 :: 200 : x$
4. Calculate: $5 \times x = 8 \times 200$
5. Solve: $x = \frac{1600}{5} = 320$.
*Result:* 320 units.

# 7. Recognition Clues & Keywords
*   **Direct Proportion:** "More... More", "Less... Less".
*   **Indirect Proportion (Do not confuse!):** "More... Less" (e.g., more workers finish a job in less time). Always check the logic before setting up the ratio.
*   **Context:** Use this for rates, unit costs, and linear scaling.

# 8. Common Mistakes
1.  **Confusing Direct with Indirect:** Students often apply the direct formula to inverse problems (like time/work). Always perform the "logic check" (If I add more workers, does time go up or down?).
2.  **Incorrect Ratio Alignment:** Placing the variables in the wrong order (e.g., $A_1/B_2$ instead of $A_1/B_1$). Ensure the units match across the numerator and denominator.
3.  **Arithmetic Errors:** Failing to simplify fractions before multiplying. Always look to divide first to keep numbers manageable.

# 9. Practice Tips
*   **The Logic Check:** Before calculating, write "More/More" or "More/Less" next to your problem to confirm the relationship type.
*   **Unit Consistency:** Ensure all time units (days/hours) and quantity units (kg/grams) are consistent before starting.
*   **Related Patterns:** Once comfortable, study **Indirect Proportion** and **Chain Rule** problems, which involve more than two variables.
