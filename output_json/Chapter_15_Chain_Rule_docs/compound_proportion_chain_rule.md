# Pattern: Compound Proportion (Chain Rule)

## 1. Pattern Overview
The Compound Proportion (Chain Rule) is a mathematical technique used to determine an unknown quantity by comparing it against multiple related variables that change in direct or inverse proportion. This pattern is essential for solving real-world problems involving work, time, resources, and costs where multiple factors influence the final outcome.

## 2. Key Formulas
The fundamental relationship for the Chain Rule is based on the comparison of ratios. If we have two sets of conditions, the relationship is expressed as:

**The General Proportion Formula:**
$$\frac{M_1 \times D_1 \times H_1}{W_1} = \frac{M_2 \times D_2 \times H_2}{W_2}$$

Where:
*   $M$ = Men (or workers/machines)
*   $D$ = Days
*   $H$ = Hours
*   $W$ = Work done

**Proportionality Rules:**
*   **Direct Proportion:** If $A \uparrow$ then $B \uparrow$, then $\frac{A_1}{A_2} = \frac{B_1}{B_2}$
*   **Indirect Proportion:** If $A \uparrow$ then $B \downarrow$, then $\frac{A_1}{A_2} = \frac{B_2}{B_1}$

## 3. When to Use This Pattern
Use this pattern when a problem provides a set of initial conditions (e.g., 10 men take 5 days to build 1 wall) and asks for a new value when one or more of those conditions change (e.g., how many days for 15 men to build 3 walls?).

*   **Keywords:** "If... then...", "How many...", "Required to...", "Same work".
*   **Recognition:** Look for problems involving workers, time, speed, distance, or consumption rates.
*   **Given vs. Found:** You are typically given a complete set of variables for one scenario and a partial set for the second; you must find the missing variable.

## 4. Core Concept & Theory
The Chain Rule works by isolating the relationship between the "unknown" variable and every other "known" variable individually. By determining if each relationship is **Direct** (as one increases, the other increases) or **Indirect** (as one increases, the other decreases), we can construct a ratio that balances the equation. The "Chain" refers to linking these ratios together to find the final value.

## 5. Step-by-Step Solution Method
1.  **Identify the Unknown:** Assign a variable (e.g., $x$) to the quantity you need to find.
2.  **List the Variables:** Write down all categories (Men, Days, Hours, Work) for both scenarios.
3.  **Compare Individually:** Compare each category to the category containing $x$.
    *   Ask: "If this variable increases, does $x$ increase or decrease?"
4.  **Determine Proportion:**
    *   If **Direct**: The ratio is $\frac{\text{Old Value}}{\text{New Value}}$.
    *   If **Indirect**: The ratio is $\frac{\text{New Value}}{\text{Old Value}}$.
5.  **Set up the Equation:** Multiply the ratios and equate them to the ratio of the unknown.
6.  **Solve for $x$:** Use algebraic manipulation to isolate $x$.

## 6. Worked Examples

**Example 1: Work and Men**
*If 36 men can do a piece of work in 25 hours, in how many hours will 15 men do it?*
*   **Given:** $M_1=36, H_1=25$; $M_2=15, H_2=x$.
*   **Reasoning:** Fewer men will take more hours (Indirect Proportion).
*   **Solution:** 
    $15 : 36 :: 25 : x$
    $15 \times x = 36 \times 25$
    $x = \frac{36 \times 25}{15} = 60$ hours.

**Example 2: Consumption**
*A canteen requires 105 kg of wheat for 7 days. How many kg will it require for 58 days?*
*   **Given:** $W_1=105, D_1=7$; $W_2=x, D_2=58$.
*   **Reasoning:** More days require more wheat (Direct Proportion).
*   **Solution:**
    $7 : 58 :: 105 : x$
    $7 \times x = 58 \times 105$
    $x = \frac{58 \times 105}{7} = 870$ kg.

## 7. Recognition Clues & Keywords
*   **Direct:** "More... More...", "Less... Less..."
*   **Indirect:** "More... Less...", "Less... More..."
*   **Avoid Confusion:** Do not confuse "Work" with "Time". Work is usually the denominator in the formula, while Time is a multiplier.

## 8. Common Mistakes
1.  **Assuming Direct Proportion for everything:** Always test the logic. If you have more workers, the time taken *must* decrease.
2.  **Inverting the Ratio:** Students often flip the ratio when it should be direct. Always check: "Does my answer make sense?"
3.  **Mixing Units:** Ensure all time units (hours/days) are consistent before starting.
4.  **Ignoring the "Work" constant:** If the work amount changes (e.g., building 2 walls instead of 1), it must be included in the ratio.

## 9. Practice Tips
*   **The "Arrow" Method:** Draw an arrow pointing from the smaller number to the larger number for each variable. If the arrows point in the same direction, it is direct; if opposite, it is indirect.
*   **Related Patterns:** Study "Time and Work" and "Speed, Distance, and Time" as they are specific applications of the Chain Rule.
