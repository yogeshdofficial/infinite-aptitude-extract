# Square Root And Cube Root — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Structure:** Scan for radicals ($\sqrt{}$, $\sqrt[3]{}$), infinite repetitions, or nested structures.
2. **Classify:** Match the structure to the Quick-Recognition Table below.
3. **Simplify Radicand:** If decimal, convert to fraction; if mixed, convert to improper; if surd, rationalize.
4. **Apply Pattern:** Use the specific formula or shortcut for the identified type.
5. **Sanity Check:** Verify the magnitude (e.g., $\sqrt{0.0009} = 0.03$, not $0.3$) and check the last digit.

## Formula Bank

**[Basic Square Roots]**
$$\sqrt{a^2 \times b^2 \times c^2} = a \times b \times c$$
→ *produces: square root of product*
→ *use when: prime factorization is possible*

$$\sqrt{\frac{a}{b}} = \frac{\sqrt{a}}{\sqrt{b}}$$
→ *produces: root of fraction*
→ *use when: radicand is a fraction*

**[Cube Roots]**
$$\sqrt[3]{a^3 \times b^3} = a \times b$$
→ *produces: cube root of product*
→ *use when: prime factorization is possible*

$$\text{Perfect cube: exponents are multiples of 3}$$
→ *produces: condition for perfect cube*
→ *use when: finding missing multiplier*

**[Decimal Square Roots]**
$$\sqrt{\frac{a}{10^{2n}}} = \frac{\sqrt{a}}{10^n}$$
→ *produces: root of decimal*
→ *use when: radicand has even decimal places*

$$\sqrt{x^2+d} \approx x + \frac{d}{2x}$$
→ *produces: approximate root*
→ *use when: number is not a perfect square*

**[General Expressions]**
$$\sqrt{a} = b \implies a = b^2$$
→ *produces: value of variable*
→ *use when: solving for unknown inside root*

$$a^2 - b^2 = (a+b)(a-b)$$
→ *produces: simplified expression*
→ *use when: difference of squares appears*

**[Infinite Series Roots]**
$$x^2 - x - n = 0$$
→ *produces: quadratic equation for $x$*
→ *use when: $\sqrt{n+\sqrt{n+...}}$*

**[Surd Simplification]**
$$x^2 + y^2 = (x+y)^2 - 2xy$$
→ *produces: sum of squares*
→ *use when: given $x$ and $y$ as conjugates*

## Step Sequences

**Basic Square Roots:** Factorize → Pair factors → Multiply pairs → Result
**Cube Roots:** Factorize → Identify missing factors → Multiply to complete triplets → Result
**Decimal Square Roots:** Count decimals → Halve count → Root digits → Place decimal
**General Expressions:** Simplify roots → Substitute variables → Solve equation → Result
**Infinite Series Roots:** Set $x = \text{expression}$ → Square both sides → Solve quadratic → Result
**Nested Roots:** Start innermost → Solve root → Add/Subtract → Repeat outward
**Surd Simplification:** Rationalize denominators → Combine terms → Simplify → Result

---

## Fastest Tricks

**Basic Square Roots:** Last digit of root matches last digit of square (except 2,3,7,8).
**Cube Roots:** Last digit of cube root matches last digit of cube (except 2↔8, 3↔7).
**Decimal Square Roots:** Decimal places in root = half of decimal places in radicand.
**Infinite Series Roots:** If $n = a(a+1)$, $\sqrt{n+\sqrt{n+...}} = a+1$.
**Nested Roots:** Always solve from the innermost radical outward.
**Surd Simplification:** $(a+\sqrt{b}) + (a-\sqrt{b}) = 2a$; $(a+\sqrt{b})(a-\sqrt{b}) = a^2-b$.

---

## Trap Watch

**Basic Square Roots:** Forgetting to pair factors → Check pairs.
**Cube Roots:** Confusing square/cube requirements → Check exponents.
**Decimal Square Roots:** Miscounting decimal places → Count pairs of digits.
**Infinite Series Roots:** Including negative root → Keep positive value.
**Nested Roots:** Solving outside-in → Solve inside-out.
**Surd Simplification:** Sign errors in subtraction → Distribute negative sign.

## Last-Minute Reminders

A number ending in 2, 3, 7, or 8 is never a perfect square.
The number of digits in $\sqrt{x}$ is $\frac{n}{2}$ (even) or $\frac{n+1}{2}$ (odd).
Always convert mixed fractions to improper fractions before taking roots.
For infinite series $\sqrt{n-\sqrt{n-...}}$, the answer is the smaller factor $a$ if $n=a(a+1)$.
Rationalizing $\frac{1}{\sqrt{a}-\sqrt{b}}$ always results in $\frac{\sqrt{a}+\sqrt{b}}{a-b}$.
