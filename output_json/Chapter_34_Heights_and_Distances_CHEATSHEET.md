# Heights And Distances — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Observations:** Count the number of distinct observation points (one point = Single; two points = Two Observations).
2. **Sketch:** Draw a right-angled triangle; label the height (vertical), base (horizontal), and hypotenuse (line of sight).
3. **Map Angles:** Mark the angle of elevation/depression; remember angle of elevation = angle of depression.
4. **Select Ratio:** Choose $\sin, \cos,$ or $\tan$ based on the sides involved (SOH-CAH-TOA).
5. **Formulate:** Set up the equation using the chosen ratio.
6. **Solve/Sanity Check:** Solve for the unknown; ensure the result is positive and physically plausible.

## Formula Bank

**[Single Observation]**

$$
\tan \theta = \frac{\text{Perpendicular}}{\text{Base}}
$$
→ *produces: height or base distance*
→ *use when: height and base are involved*

$$
\sin \theta = \frac{\text{Opposite}}{\text{Hypotenuse}}
$$
→ *produces: height or ladder length*
→ *use when: hypotenuse is involved*

$$
\cos \theta = \frac{\text{Adjacent}}{\text{Hypotenuse}}
$$
→ *produces: base or ladder length*
→ *use when: base and hypotenuse involved*

**[Two Observations]**

$$
h = d \times \frac{\tan \alpha \tan \beta}{\tan \beta - \tan \alpha}
$$
→ *produces: height of tower*
→ *use when: moving between two angles*

$$
\text{Time} = \frac{\text{Distance}}{\text{Speed}}
$$
→ *produces: time taken for travel*
→ *use when: constant speed boat/object*

$$
\tan(90^{\circ} - \theta) = \cot \theta
$$
→ *produces: complementary angle ratio*
→ *use when: angles sum to $90^{\circ}$*

## Fastest Tricks

* **[Single Observation]:** If $\theta = 45^{\circ}$, height equals base.
* **[Single Observation]:** If $\theta = 30^{\circ}$, base = height $\times \sqrt{3}$.
* **[Single Observation]:** If $\theta = 60^{\circ}$, height = base $\times \sqrt{3}$.
* **[Two Observations]:** For $30^{\circ}$ to $60^{\circ}$ movement, height = $\frac{\sqrt{3}}{2} \times \text{distance moved}$.

---

## Trap Watch

* **[Single Observation]:** Confusing $\tan 30^{\circ}$ and $\tan 60^{\circ}$ → check table values.
* **[Two Observations]:** Forgetting to add distance moved → add to base.
* **[Two Observations]:** Mixing elevation and depression → they are equal.

---

## Last-Minute Reminders

Always verify if the observer's height must be added to the final result.
Remember that $\tan 30^{\circ} = \frac{1}{\sqrt{3}}$ and $\tan 60^{\circ} = \sqrt{3}$.
Ensure your calculator or mental math is in degrees, not radians.
The hypotenuse is always the longest side of the right-angled triangle.
Angle of depression is measured from the horizontal line of sight downwards.
