# Simplification — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Type:** Scan for variables (Algebraic), nested operations (Arithmetic), custom symbols (Custom), or repeating patterns (Series).
2. **Map Pattern:** Use the Quick-Recognition Table to match the question structure to a specific category.
3. **Isolate Variables:** Use substitution, ratio assignment ($x=ak$), or BODMAS to simplify the expression into a solvable form.
4. **Apply Formula:** Select the relevant identity or series property from the Formula Bank.
5. **Execute Calculation:** Perform arithmetic step-by-step, respecting order of operations (brackets first).
6. **Sanity Check:** Verify signs, check if the magnitude is logical, and ensure the result matches the provided options.

## Formula Bank

**[Algebraic Identities]**
$(x+y)^2 = x^2 + y^2 + 2xy$ → *gives sum of squares* → *use when $x+y$ and $xy$ known*
$(x-y)^2 = x^2 + y^2 - 2xy$ → *gives difference of squares* → *use when $x-y$ and $xy$ known*
$a^3+b^3+c^3=3abc \implies a+b+c=0$ → *solves for variable* → *use when sum of cubes equals $3abc$*
$\frac{a}{b} = \frac{c}{d} \implies \frac{a+b}{a-b} = \frac{c+d}{c-d}$ → *simplifies ratios* → *use for Componendo and Dividendo*

**[Arithmetic Simplification]**
$a \div b \div c = \frac{a}{b \times c}$ → *corrects division order* → *use for multiple divisions*
$a^2 - b^2 = (a+b)(a-b)$ → *factors expression* → *use for difference of squares*
$a^3 - b^3 = (a-b)(a^2+ab+b^2)$ → *factors expression* → *use for difference of cubes*

**[Custom Operations]**
$x * y = \text{rule provided}$ → *evaluates expression* → *use for custom symbols*

**[Series and Sequences]**
$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$ → *telescoping sum* → *use for partial fractions*
$(1 - \frac{1}{n}) = \frac{n-1}{n}$ → *telescoping product* → *use for sequential products*

**[Uncategorized]**
$a^n \pmod{m}$ → *finds remainder* → *use for large powers*

**[Word Problems]**
$\text{Total} = \text{Part}_1 + \text{Part}_2$ → *solves systems* → *use for linear equations*

## Step Sequences

**Algebraic:** Assign ratio $k$ → Substitute into identity → Solve for $k$ → Calculate.
**Arithmetic:** Apply BODMAS → Simplify brackets → Perform division/multiplication → Add/Subtract.
**Custom:** Identify $x, y$ values → Apply rule → Solve inner brackets → Final result.
**Series:** Factor denominator → Apply telescoping rule → Cancel intermediate terms → Simplify.
**Uncategorized:** Calculate first 3 powers → Identify remainder cycle → Apply to $n$ → Result.
**Word Problems:** Define variables → Form linear equations → Eliminate/Substitute → Solve.

---

## Fastest Tricks

* **Algebraic:** Use $x^2+y^2 = (x+y)^2 - 2xy$ to skip expansion.
* **Arithmetic:** For $a \div b \div c$, always write as $\frac{a}{b \times c}$.
* **Series:** Sum is $\frac{1}{\text{first factor}} - \frac{1}{\text{last factor}}$.
* **Word Problems:** For $x^2+bx+c=0$, roots sum to $-b$ and multiply to $c$.
* **Uncategorized:** For $4^n \div 6$, remainder is always $4$ for $n \ge 1$.

---

## Trap Watch

* **Algebraic:** Forgetting to multiply $xy$ by $2$ → Always double $xy$.
* **Arithmetic:** Dividing left-to-right incorrectly → Use fraction form.
* **Custom:** Mixing up $x$ and $y$ order → Label variables clearly.
* **Series:** Misidentifying first/last terms → Write out first two terms.
* **Word Problems:** Sign errors in subtraction → Use brackets for equations.

---

## Last-Minute Reminders

Always perform operations inside the innermost brackets first.
When dealing with ratios like $\frac{a}{b} = \frac{4}{5}$, always use $a=4k, b=5k$.
In telescoping series, ensure the numerator difference matches the denominator difference.
Check if the question asks for $x$ or a function of $x$ like $x^2+y^2$.
When dividing by 6, powers of 4 always leave a remainder of 4.
