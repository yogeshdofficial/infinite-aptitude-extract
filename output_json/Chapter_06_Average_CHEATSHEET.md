# Average — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Goal:** Determine if you need a single missing value, a new average after a change, or a combined weighted average.
2. **Map to Pattern:** Use the Quick-Recognition Table to match the question's structure to a specific pattern.
3. **Extract Totals:** Convert all averages to sums immediately ($\text{Sum} = \text{Average} \times \text{Count}$).
4. **Set Up Equation:** Use the sum-based equation to isolate the unknown variable.
5. **Apply Shortcut:** Check if Alligation or Deviation methods apply to bypass long arithmetic.
6. **Sanity Check:** Ensure the result is within the range of the given values; check units (e.g., months vs. years).

## Formula Bank

**[Basic Average]**
$$\text{Average} = \frac{\sum x_i}{n}$$
→ *produces: arithmetic mean of set*
→ *use when: simple list provided*

$$\text{Average of AP} = \frac{\text{First} + \text{Last}}{2}$$
→ *produces: mean of sequence*
→ *use when: numbers are in AP*

**[Conditional Average]**
$$\text{Sum} = \text{Average} \times \text{Count}$$
→ *produces: total sum of set*
→ *use when: finding missing total*

**[Missing Value]**
$$\text{Total} = \text{Average} \times \text{Count}$$
→ *produces: total sum of items*
→ *use when: one value is missing*

**[Property Transformation]**
$$(a+b+c)^2 = a^2+b^2+c^2+2(ab+bc+ca)$$
→ *produces: sum of squares*
→ *use when: mean of squares needed*

**[Replacement and Change]**
$$\text{New Total} = \text{Old Total} - \text{Out} + \text{In}$$
→ *produces: updated sum*
→ *use when: members are replaced*

$$\text{New Avg} = \text{Old Avg} + \frac{\text{Diff}}{\text{Count}}$$
→ *produces: updated average*
→ *use when: value added/removed*

**[Uncategorized]**
$$\text{Avg Speed} = \frac{2xy}{x+y}$$
→ *produces: average speed*
→ *use when: equal distance traveled*

**[Weighted Average]**
$$\text{Weighted Avg} = \frac{\sum (x_i \times f_i)}{\sum f_i}$$
→ *produces: combined average*
→ *use when: multiple groups exist*

## Step Sequences

**Basic Average:** Sum terms → Count items → Divide sum by count → Answer
**Conditional Average:** Calculate mean → Compare values → Count matches → Answer
**Missing Value:** Calculate total sum → Subtract knowns → Result is missing → Answer
**Property Transformation:** Identify relation → Apply identity → Solve for target → Answer
**Replacement/Change:** Find old total → Adjust for change → Divide by count → Answer
**Uncategorized:** Convert ratio to sum → Apply formula → Solve for variable → Answer
**Weighted Average:** Find group sums → Add sums → Divide by total count → Answer

---

## Fastest Tricks

* **Basic Average:** Use deviation from an assumed mean to simplify addition.
* **Replacement:** New value = Old Avg + (Change in Avg $\times$ Total Count).
* **Weighted Average:** Use Alligation for two groups to find ratio of counts.
* **AP Average:** Always use $\frac{\text{First} + \text{Last}}{2}$ for consecutive numbers.
* **Average Speed:** Use $\frac{2xy}{x+y}$ only if distances are identical.

---

## Trap Watch

* **Basic Average:** Miscounting items → Verify count manually.
* **Conditional Average:** Ignoring zero/negative values → Check signs carefully.
* **Missing Value:** Forgetting to multiply average by count → Always find total.
* **Replacement:** Confusing months/years → Convert all to same unit.
* **Weighted Average:** Averaging the averages → Always use weighted sum.

## Last-Minute Reminders

The average of a set of numbers is always between the smallest and largest values.
If every number in a set increases by $k$, the average also increases by $k$.
If every number is multiplied by $k$, the average is also multiplied by $k$.
Average speed is not the arithmetic mean of speeds unless time spent is equal.
When replacing members, the change in total is (New - Old) $\times$ Count.
