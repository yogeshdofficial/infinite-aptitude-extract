# Decimal Fractions — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Type:** Scan for symbols ($, \sqrt{}, \overline{x}$), structure (fractions, decimals), or keywords (ascending, scientific, product).
2. **Classify:** Match the structure to the Quick-Recognition Table below.
3. **Standardize:** Convert all mixed formats (fractions/decimals) into a single format (usually decimals for ordering, fractions for recurring).
4. **Apply Identity/Formula:** Use the specific formula from the Bank. If algebraic, look for $a^2-b^2$ or $a^3 \pm b^3$ patterns.
5. **Execute:** Perform arithmetic using column alignment or decimal shift rules.
6. **Sanity Check:** Verify decimal placement by counting total places in factors vs. result.

## Formula Bank

**[Algebraic Simplification]**
$$a^2 - b^2 = (a + b)(a - b)$$
→ *produces: simplified numerator/denominator*
→ *use when: difference of two squares appears*

$$a^3 \pm b^3 = (a \pm b)(a^2 \mp ab + b^2)$$
→ *produces: cancellation of complex terms*
→ *use when: sum or difference of cubes*

$$(a \pm b)^2 = a^2 \pm 2ab + b^2$$
→ *produces: expanded binomial*
→ *use when: squaring a decimal sum/diff*

**[Decimal Arithmetic]**
$$\text{Total decimal places} = \sum \text{decimal places of factors}$$
→ *produces: correct decimal placement*
→ *use when: multiplying decimals*

$$\frac{a}{b} = \frac{a \times 10^n}{b \times 10^n}$$
→ *produces: integer division*
→ *use when: dividing decimals*

**[Expression Equivalence]**
$$\text{Net shift} = \sum \text{num. decimals} - \sum \text{den. decimals}$$
→ *produces: decimal point position*
→ *use when: comparing complex expressions*

**[Fraction Conversion]**
$$\text{Decimal} = \frac{\text{Number without decimal}}{10^n}$$
→ *produces: vulgar fraction*
→ *use when: converting terminating decimals*

**[Fraction Ordering]**
$$\text{Value} = \text{Numerator} \div \text{Denominator}$$
→ *produces: decimal magnitude*
→ *use when: comparing multiple fractions*

**[Recurring Decimals]**
$$0.\overline{x} = \frac{x}{10^n - 1}$$
→ *produces: fraction for pure recurring*
→ *use when: repeating digits start immediately*

$$0.a\overline{bc} = \frac{abc - a}{990}$$
→ *produces: fraction for mixed recurring*
→ *use when: non-repeating digits exist*

**[Scientific Notation]**
$$a \times 10^n, 1 \le a < 10$$
→ *produces: scientific form*
→ *use when: standardizing large/small numbers*

## Step Sequences

**Algebraic Simplification:** Identify identity → Substitute values → Simplify terms → Calculate result
**Decimal Arithmetic:** Align points → Pad zeros → Add/Subtract columns → Place point
**Expression Equivalence:** Map digits to variables → Solve for variables → Substitute → Compute
**Fraction Conversion:** Count decimal places → Write over $10^n$ → Simplify fraction → Final form
**Fraction Ordering:** Convert to decimals → Compare magnitudes → Sort list → Final order
**Recurring Decimals:** Identify repeating part → Apply $9/90$ rule → Simplify fraction → Final form
**Scientific Notation:** Move decimal → Count shifts → Set exponent → Write form
**Word Problems:** Define variables → Form equation → Solve for $k$ → Calculate values

---

## Fastest Tricks

**Algebraic:** Treat decimals as integers, then adjust decimal places at the end.
**Fraction Ordering:** For $\frac{n}{n+1}$ forms, the largest fraction has the largest $n$.
**Recurring:** $0.\overline{142857} = \frac{1}{7}$.
**Scientific:** $10^m \times 10^n = 10^{m+n}$; add exponents for multiplication.
**Arithmetic:** $a - b - c = a - (b + c)$; sum subtrahends first.

---

## Trap Watch

**Algebraic:** Forgetting $2ab$ term → include middle term.
**Decimal Arithmetic:** Misaligning decimal points → use grid paper.
**Recurring:** Using $100$ instead of $99$ → use $9$s for repeating.
**Scientific:** Confusing positive/negative exponents → right shift is negative.
**Word Problems:** Confusing Rupees/Paise → convert all to same unit.

---

## Last-Minute Reminders

Always align decimal points vertically before adding or subtracting.
The number of decimal places in a product is the sum of decimal places in the factors.
For recurring decimals, the number of nines in the denominator equals the number of repeating digits.
When dividing by a decimal, multiply both numerator and denominator by $10^n$ to remove the decimal.
In scientific notation, the coefficient must be between $1$ and $10$.
