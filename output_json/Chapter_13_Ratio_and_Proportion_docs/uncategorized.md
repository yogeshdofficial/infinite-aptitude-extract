# 1. Pattern Overview
This pattern provides a comprehensive framework for analyzing relationships between quantities through ratios, proportions, and variations. It is essential for solving problems involving scaling, mixtures, comparisons, and proportional dependencies in real-world scenarios.

# 2. Key Formulas

**Ratio and Proportion**
*   **Ratio:** $a : b = \frac{a}{b}$
*   **Proportion:** If $a : b :: c : d$, then $bc = ad$
*   **Fourth Proportional ($d$):** $d = \frac{bc}{a}$
*   **Third Proportional ($c$):** $c = \frac{b^2}{a}$
*   **Mean Proportional:** $\sqrt{ab}$

**Ratio Operations**
*   **Compounded Ratio:** $(a:b), (c:d), (e:f) \rightarrow (ace : bdf)$
*   **Duplicate Ratio:** $a^2 : b^2$
*   **Sub-duplicate Ratio:** $\sqrt{a} : \sqrt{b}$
*   **Triplicate Ratio:** $a^3 : b^3$
*   **Sub-triplicate Ratio:** $\sqrt[3]{a} : \sqrt[3]{b}$
*   **Componendo and Dividendo:** $\frac{a+b}{a-b} = \frac{c+d}{c-d}$

**Variation and Mixtures**
*   **Direct Variation:** $x = ky$ (where $k$ is a constant)
*   **Inverse Variation:** $xy = k$
*   **Mixture Replacement:** $\text{Final Liquid} = x \left(1 - \frac{y}{x}\right)^n$

# 3. When to Use This Pattern
*   **Keywords:** "Ratio of," "Proportional to," "Directly/Inversely as," "Mean proportional," "Replaced by," "Compounded."
*   **Recognition:** Use when comparing two or more quantities, finding a missing term in a sequence of proportions, or calculating the concentration of a substance after repeated dilution.
*   **Given/Found:** Typically, you are given initial values or a relationship constant ($k$) and asked to find a missing term or the final state of a mixture.

# 4. Core Concept & Theory
The core of this pattern is the **invariance of ratios**. Whether you multiply or divide terms by a non-zero constant, the relationship remains identical. Proportions extend this by equating two ratios, allowing us to solve for unknown variables using the "Product of Means = Product of Extremes" rule. Variation theory allows us to model physical laws where one variable depends on another.

# 5. Step-by-Step Solution Method
1.  **Identify the Relationship:** Determine if the problem involves a simple ratio, a proportion, or a variation.
2.  **Standardize Units:** Ensure all quantities are in the same units before forming the ratio.
3.  **Set up the Equation:** Translate the word problem into a mathematical expression using the formulas above.
4.  **Apply Algebraic Manipulation:** Use cross-multiplication or Componendo/Dividendo to isolate the unknown variable.
5.  **Verify:** Check if the result satisfies the original ratio condition (e.g., $bc = ad$).

# 6. Worked Examples

**Example 1: Finding the Fourth Proportional**
Given: Find the fourth proportional to 4, 9, and 12.
Solution:
1. Let the fourth proportional be $d$.
2. The proportion is $4 : 9 :: 12 : d$.
3. Using $bc = ad$: $9 \times 12 = 4 \times d$.
4. $108 = 4d \rightarrow d = 27$.

**Example 2: Mixture Replacement**
Given: A container has 50L of milk. 5L is removed and replaced with water. This is done 2 times. Find the final amount of milk.
Solution:
1. Use the formula: $x(1 - y/x)^n$ where $x=50, y=5, n=2$.
2. $50(1 - 5/50)^2 = 50(1 - 0.1)^2$.
3. $50(0.9)^2 = 50(0.81) = 40.5$ units.

# 7. Recognition Clues & Keywords
*   **"Directly proportional":** $x/y = k$.
*   **"Inversely proportional":** $xy = k$.
*   **"Mean proportional":** Look for the square root of the product of two numbers.
*   **"Replaced":** Signals the mixture replacement formula.
*   **Avoid confusion:** Do not confuse "Compounded ratio" (multiplication of ratios) with "Proportion" (equality of ratios).

# 8. Common Mistakes
1.  **Unit Mismatch:** Trying to form a ratio between different units (e.g., meters and centimeters). Always convert to the same unit first.
2.  **Inverse vs. Direct:** Swapping the formulas for direct and inverse variation. Remember: Direct = Division constant ($x/y=k$), Inverse = Multiplication constant ($xy=k$).
3.  **Mixture Formula Error:** Forgetting that $n$ represents the number of *operations*, not the total volume.
4.  **Order of Terms:** In $a:b :: c:d$, ensure the order is strictly followed; $a$ and $d$ are extremes, $b$ and $c$ are means.

# 9. Practice Tips
*   **Master the Basics:** Practice simplifying ratios to their lowest terms first.
*   **Visualize:** For variation problems, draw a quick table to see if as one value increases, the other increases (direct) or decreases (inverse).
*   **Related Patterns:** Study "Percentage" and "Time and Work," as they rely heavily on the foundational principles of ratio and proportion.
