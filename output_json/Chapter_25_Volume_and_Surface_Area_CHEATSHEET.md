# Volume And Surface Area — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Geometry:** Scan for keywords (e.g., "cone," "cube," "melted," "flow rate").
2. **Standardize Units:** Convert all dimensions to the same unit (e.g., cm to m) *before* calculating.
3. **Select Formula:** Choose the formula based on the required output (Volume vs. Surface Area).
4. **Equate/Solve:** If "melted" or "recast," equate volumes ($V_1 = V_2$). If "flow," use $V = \text{Area} \times \text{Velocity} \times \text{Time}$.
5. **Sanity Check:** Ensure the answer is positive and units match the question requirements.

## Formula Bank

**[Cone Geometry]**
$$l = \sqrt{r^2 + h^2}$$ → *slant height* → *when $h$ and $r$ given*
$$V = \frac{1}{3} \pi r^2 h$$ → *volume* → *capacity of cone*
$$\text{CSA} = \pi rl$$ → *curved surface area* → *lateral area only*
$$\text{TSA} = \pi rl + \pi r^2$$ → *total surface area* → *including base*

**[Cube Geometry]**
$$d = a\sqrt{3}$$ → *space diagonal* → *diagonal from edge*
$$V = a^3$$ → *volume* → *capacity of cube*
$$\text{SA} = 6a^2$$ → *surface area* → *total area*

**[Cuboid Geometry]**
$$V = l \times b \times h$$ → *volume* → *capacity of cuboid*
$$\text{SA} = 2(lb + bh + lh)$$ → *surface area* → *total area*
$$d = \sqrt{l^2 + b^2 + h^2}$$ → *space diagonal* → *longest pole length*

**[Cylinder Geometry]**
$$V = \pi r^2 h$$ → *volume* → *capacity of cylinder*
$$\text{CSA} = 2\pi rh$$ → *curved surface area* → *lateral area*
$$\text{TSA} = 2\pi r(h+r)$$ → *total surface area* → *including two bases*

**[Pyramid Geometry]**
$$V = \frac{1}{3} \times \text{Base Area} \times h$$ → *volume* → *capacity of pyramid*

**[Sphere Geometry]**
$$V = \frac{4}{3}\pi r^3$$ → *volume of sphere* → *capacity of sphere*
$$V = \frac{2}{3}\pi r^3$$ → *volume of hemisphere* → *capacity of hemisphere*
$$\text{CSA} = 2\pi r^2$$ → *hemisphere CSA* → *curved area*
$$\text{TSA} = 3\pi r^2$$ → *hemisphere TSA* → *including flat base*

## Step Sequences
* **Cone:** Find $l$ → Apply $V$ or $\text{SA}$ formula → Adjust for count.
* **Cube:** Find $a$ → Apply $V$ or $\text{SA}$ formula → Check units.
* **Cuboid:** Identify $l, b, h$ → Apply $V$ or $\text{SA}$ formula → Solve.
* **Cylinder:** Find $r, h$ → Apply $V$ or $\text{SA}$ formula → Convert units.
* **Pyramid:** Find Base Area → Multiply by $h/3$ → Answer.
* **Sphere:** Find $r$ → Apply $V$ or $\text{SA}$ formula → Adjust for hemisphere.
* **Misc:** Calculate total $V$ → Subtract waste/mortar → Divide by unit $V$.

---

## Fastest Tricks
* **Cone:** Use $3:4:5$ ratio for $r, h, l$ to find slant height instantly.
* **Cube:** If edge increases by $k$, area increases by $k^2$, volume by $k^3$.
* **Cuboid:** Longest pole is $\sqrt{l^2+b^2+h^2}$; use Pythagorean triples.
* **Cylinder:** $V = \text{Base Area} \times h$; use $\pi r^2$ as a single block.
* **Sphere:** $V_{\text{sphere}} : V_{\text{cylinder}} : V_{\text{cone}} = 4:6:2$ (for same $r, h=2r$).
* **Misc:** $1 \text{ m}^3 = 1000 \text{ litres}$; $1 \text{ litre} = 1000 \text{ cm}^3$.

## Trap Watch
* **Cone:** Using $h$ instead of $l$ for $\text{SA}$ → Use $\sqrt{r^2+h^2}$.
* **Cube:** Forgetting to divide by 6 before $\sqrt{\text{SA}}$ → Divide first.
* **Cuboid:** Confusing face diagonal with space diagonal → Use $\sqrt{l^2+b^2+h^2}$.
* **Cylinder:** Using diameter instead of radius → Divide by 2.
* **Sphere:** Using full sphere formula for hemisphere → Use $\frac{2}{3}\pi r^3$.
* **Misc:** Forgetting to subtract mortar/waste volume → Subtract percentage first.

---

## Last-Minute Reminders
Always convert all dimensions to the same unit (cm or m) before starting calculations.
Total Surface Area of a cylinder must include the area of both circular bases ($2\pi r^2$).
The volume of a pyramid is always exactly one-third of the volume of a prism with the same base and height.
When melting objects, the total volume remains constant regardless of the shape change.
A hemisphere's total surface area includes the flat circular top, making it $3\pi r^2$, not $2\pi r^2$.
