# Time And Work — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Actors:** List all individuals/groups and their given time or rate.
2. **Standardize:** Convert all "fraction of work" or "efficiency ratio" data into a common unit (usually daily rate or total work units).
3. **Select Pattern:** Match the question structure to the Quick-Recognition Table below.
4. **Formulate:** Use the Formula Bank to set up an equation (e.g., $\sum \text{Rates} = \text{Total Rate}$).
5. **Solve:** Isolate the unknown variable.
6. **Sanity Check:** Ensure the time taken is positive and logically consistent with the individual rates.

## Formula Bank

**[Alternate Days]**
$$\text{Work in cycle} = \sum \text{Rates of cycle}$$
→ *produces: work done per full cycle*
→ *use when: workers alternate daily*

**[Basic Work Together]**
$$\text{Time} = \frac{xy}{x+y}$$
→ *produces: combined time for two*
→ *use when: two people work together*

$$\text{Time} = \frac{1}{\sum \frac{1}{t_i}}$$
→ *produces: combined time for many*
→ *use when: three or more people*

**[Efficiency and Ratio]**
$$\text{Time}_A \times \text{Efficiency}_A = \text{Time}_B \times \text{Efficiency}_B$$
→ *produces: missing time or efficiency*
→ *use when: comparing two workers*

**[Find Individual Time]**
$$\text{Rate}_A = \text{Rate}_{A+B} - \text{Rate}_B$$
→ *produces: individual rate*
→ *use when: combined rate is known*

**[Fractional Work]**
$$\text{Total Time} = \frac{\text{Time taken}}{\text{Fraction of work}}$$
→ *produces: time for full work*
→ *use when: partial work is given*

**[Group Work]**
$$\frac{M_1 D_1 H_1}{W_1} = \frac{M_2 D_2 H_2}{W_2}$$
→ *produces: missing variable*
→ *use when: group size/hours change*

**[Uncategorized]**
$$T = \sqrt{xy}$$
→ *produces: combined time T*
→ *use when: A takes T+x, B takes T+y*

**[Wages and Shares]**
$$\text{Share} = \frac{\text{Work Done}}{\text{Total Work}} \times \text{Total Wages}$$
→ *produces: individual payment*
→ *use when: distributing total earnings*

**[Work Interruption]**
$$\text{Time} = \frac{\text{Remaining Work}}{\text{Rate of remaining workers}}$$
→ *produces: time to finish*
→ *use when: someone leaves mid-task*

## Step Sequences
- **Alternate Days:** Calculate cycle work → Divide total by cycle → Add remainder.
- **Basic Work Together:** Find individual rates → Sum rates → Take reciprocal.
- **Efficiency and Ratio:** Convert efficiency to rate → Find total work → Divide.
- **Find Individual Time:** Find combined rate → Subtract known rate → Reciprocal.
- **Fractional Work:** Find full time for each → Sum rates → Reciprocal.
- **Group Work:** Equate work units → Solve for one unit → Calculate total.
- **Uncategorized:** Identify x and y → Apply square root formula.
- **Wages and Shares:** Calculate work ratio → Divide total by ratio.
- **Work Interruption:** Subtract completed work → Divide remainder by rate.

---

## Fastest Tricks
- **Alternate Days:** Cycle work $\times$ cycles $\approx$ total work; handle remainder manually.
- **Basic Work Together:** For two people, use $\frac{\text{Product}}{\text{Sum}}$ for instant time.
- **Efficiency:** Time ratio is the inverse of efficiency ratio.
- **Uncategorized:** If A takes $x$ days more and B takes $y$ days more than combined, $T = \sqrt{xy}$.
- **Wages:** Share is always proportional to work done, not time taken.

---

## Trap Watch
- **Alternate Days:** Cycle is 2 days → multiply by 2.
- **Basic Work Together:** Adding days instead of rates → add reciprocals.
- **Efficiency:** Confusing time ratio with efficiency ratio → invert ratio.
- **Work Interruption:** Forgetting to subtract work done → subtract first.
- **Wages:** Distributing by time instead of work → use work ratio.

---

## Last-Minute Reminders
- Always convert mixed fractions to improper fractions before calculating rates.
- If a worker leaves, the remaining work must be completed by the remaining staff.
- Total work can be assumed as the LCM of all given time values to simplify math.
- Wages are always distributed based on the actual work performed by each individual.
- When multiple groups work, ensure all units (men, women, children) are converted to a single base efficiency.
