# Ratio and Proportion

## Overview
These questions involve quantities related by a fixed ratio. The central idea is to represent each quantity as a multiple of a common variable (e.g., $3x, 4x$) to convert ratio relationships into a single algebraic equation.

---

## Recognition Clues
* **Keywords:** "Ratio of", "in the ratio", "divided into parts".
* **Given:** A ratio (e.g., $3:4:6$) and a condition (sum, product, average, or a change in value).
* **Asked:** The value of one or all of the individual parts.

---

## Key Formulas

### [Common Multiplier Method]
$$
\text{Value}_n = \text{Ratio}_n \times x
$$
**Variables:**
- $\text{Value}_n$ = The actual numerical value of the $n^{th}$ part.
- $\text{Ratio}_n$ = The $n^{th}$ term of the given ratio.
- $x$ = The common multiplier (the unknown we solve for).

**When to use:** Every question in this pattern.

**Worked example:** If numbers are in ratio $3:2:5$ and sum of squares is $1862$:
$(3x)^2 + (2x)^2 + (5x)^2 = 1862 \Rightarrow 38x^2 = 1862 \Rightarrow x^2 = 49 \Rightarrow x = 7$.

---

## Solution Framework
1. **Assign Variables:** Represent the quantities as $ax, bx, cx...$ using the given ratio.
2. **Form Equation:** Translate the problem statement (sum, product, average, etc.) into an equation using these variables.
3. **Solve for $x$:** Perform algebraic operations to isolate $x$ (or $x^2, x^3$).
4. **Calculate Parts:** Multiply the found $x$ by the original ratio terms to get the final answers.

---

## Shortcut Tricks
* **Trick:** For "increased by" problems (e.g., Q41), use the difference in ratio parts.
* **Why it works:** If $4x+4$ and $7x+4$ result in $3:5$, the gap between ratio parts is proportional to the constant added.
* **When to use:** When the same amount is added to both terms of a ratio.
* **Example:** Ratio $4:7 \to 3:5$. Difference in parts: $(7-4)=3$ and $(5-3)=2$. Normalize the difference or cross-multiply: $5(4x+4) = 3(7x+4) \Rightarrow x=8$.

---

## Common Mistakes
* **Mistake:** Squaring/cubing the ratio terms without the variable $x$.
    * **Fix:** Always write $(ax)^2$ or $(ax)^3$.
* **Mistake:** Adding the constant to the ratio instead of the variable term.
    * **Fix:** Use $(ax + k)$ where $k$ is the added value.
* **Mistake:** Forgetting to divide by the number of terms when "average" is mentioned.
    * **Fix:** $\text{Average} = \frac{\sum \text{terms}}{n}$.

---

## Worked Example (Step-by-Step)

**Question:** Two numbers are in the ratio $4:7$. If each is increased by $4$, the ratio becomes $3:5$. The larger number is?

**Solution:**
1. **Assign Variables:** Let the numbers be $4x$ and $7x$.
2. **Form Equation:** $\frac{4x+4}{7x+4} = \frac{3}{5}$.
3. **Solve for $x$:** Cross-multiply: $5(4x+4) = 3(7x+4) \Rightarrow 20x + 20 = 21x + 12 \Rightarrow x = 8$.
4. **Calculate Parts:** Larger number is $7x = 7 \times 8 = 56$.

**Answer:** 56

---

## Similar Patterns
**Algebraic Word Problems:** These are often confused with Ratio problems. Distinguish them by checking if a fixed ratio is explicitly provided; if not, use standard algebraic variables ($x, y$).

---

## Revision Summary
* **Key formula:** $\text{Value} = \text{Ratio} \times x$.
* **Spot it by:** Looking for "ratio of" and a total/product/average condition.
* **First move:** Assign $ax, bx, cx$ to the quantities.
* **Fastest method:** Cross-multiplication for ratio changes.
* **Biggest trap:** Forgetting to apply the exponent to the variable $x$.

---

## Appendix: Prerequisites

### Concepts You Must Know
* **Algebraic Equations** — Balancing both sides of an equation by performing the same operation.
* **Average** — The sum of values divided by the count of values: $\text{Avg} = \frac{\sum x}{n}$.
* **Cross-Multiplication** — If $\frac{a}{b} = \frac{c}{d}$, then $ad = bc$.

### Formulas You Must Know First
* **Sum of Squares:** $(ax)^2 = a^2x^2$.
* **Product of Terms:** $(ax)(bx)(cx) = (abc)x^3$.

### Terms Used In This Pattern
* **Common Multiplier ($x$):** The factor that scales the ratio to the actual values.
* **Consecutive:** Numbers following each other in order (e.g., $x, x+1$).
