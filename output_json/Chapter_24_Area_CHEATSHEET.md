# Area — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Geometry:** Scan for keywords (circle, square, rectangle, rhombus, triangle, path, cross-road).
2. **Extract Dimensions:** List given values; convert all units to a common standard (e.g., cm to m, hectares to $m^2$).
3. **Select Pattern:** Match the geometry to the Quick-Recognition Table.
4. **Formulate:** Write the relevant formula; if unknown variables exist, set up an equation using the provided area or perimeter.
5. **Execute:** Solve for the missing variable; use shortcuts if applicable.
6. **Sanity Check:** Verify units, check for logical magnitude, and ensure the answer matches the question's specific request (e.g., diagonal vs. side).

## Formula Bank

**[Circular Geometry]**
$$Area = \pi r^2$$
→ *produces: total surface area*
→ *use when: given radius or diameter*

$$Circumference = 2\pi r$$
→ *produces: perimeter of circle*
→ *use when: calculating fencing costs*

$$d = \sqrt{l^2 + b^2}$$
→ *produces: diagonal/max distance*
→ *use when: rectangle corners*

**[Parallelogram and Rhombus]**
$$Area = base \times height$$
→ *produces: area of parallelogram*
→ *use when: base and height known*

$$Area = \frac{1}{2} \times d_1 \times d_2$$
→ *produces: area of rhombus*
→ *use when: both diagonals known*

$$Area = \frac{1}{2} \times (a + b) \times h$$
→ *produces: area of trapezium*
→ *use when: parallel sides known*

**[Rectangle Basics]**
$$Area = l \times b$$
→ *produces: surface area*
→ *use when: length and breadth known*

$$Perimeter = 2(l + b)$$
→ *produces: boundary length*
→ *use when: fencing or path*

**[Square Properties]**
$$Area = side^2$$
→ *produces: area of square*
→ *use when: side length known*

$$Area = \frac{1}{2} \times diagonal^2$$
→ *produces: area of square*
→ *use when: diagonal length known*

**[Triangular Geometry]**
$$Area = \frac{1}{2} \times base \times height$$
→ *produces: area of triangle*
→ *use when: base and height known*

$$s = \frac{a+b+c}{2}; Area = \sqrt{s(s-a)(s-b)(s-c)}$$
→ *produces: area of triangle*
→ *use when: three sides known*

**[Uncategorized]**
$$Area_{path} = x(L+B-x)$$
→ *produces: area of cross-roads*
→ *use when: roads intersect middle*

$$Net Change = x + y + \frac{xy}{100}$$
→ *produces: percentage error/change*
→ *use when: dimensions change*

**[Volume and Surface Area]**
$$Total Surface Area = 2\pi r(h+r)$$
→ *produces: cylinder surface area*
→ *use when: cylinder dimensions given*

$$Volume = a^3; TSA = 6a^2$$
→ *produces: cube volume/surface*
→ *use when: cube dimensions given*

## Step Sequences

**Circular:** Find radius → Apply $\pi r^2$ → Multiply by rate.
**Parallelogram/Rhombus:** Identify base/height → Apply formula → Solve for variable.
**Rectangle:** Find $l, b$ → Apply $l \times b$ → Calculate cost.
**Square:** Find side/diagonal → Apply area formula → Calculate perimeter.
**Triangular:** Find $s$ → Apply Heron's → Calculate area.
**Uncategorized:** Identify change/path → Apply formula → Solve equation.
**Volume/Surface:** Convert units → Apply formula → Calculate result.

## Fastest Tricks

* **Circular:** Ratio of areas = square of ratio of radii.
* **Rectangle:** If area constant, $l \uparrow x\% \implies b \downarrow \frac{x}{100+x} \times 100\%$.
* **Square:** $Area \propto diagonal^2$; use $x+y+\frac{xy}{100}$ for percentage changes.
* **Rhombus:** Half-diagonals and side form a right triangle ($a^2 = (\frac{d_1}{2})^2 + (\frac{d_2}{2})^2$).
* **Triangular:** Use Pythagorean triples (e.g., 3-4-5, 5-12-13) to find missing sides instantly.

## Trap Watch

* **Circular:** Using diameter instead of radius → divide by 2.
* **Rectangle:** Forgetting to convert hectares to $m^2$ → multiply by 10,000.
* **Rhombus:** Using full diagonal in Pythagoras → use half-diagonal.
* **Uncategorized:** Forgetting to subtract intersection area ($x^2$) → subtract $x^2$.
* **Volume:** Confusing mm with cm → convert units first.

---

## Last-Minute Reminders

1. Always verify if the question asks for perimeter or area before calculating.
2. Remember that $1 \text{ hectare} = 10,000 \text{ m}^2$.
3. When calculating percentage change, always use the original value as the denominator.
4. In cross-road problems, the intersection area $x^2$ must be subtracted from the total path area.
5. Always check if the units of the rate (e.g., per $m^2$) match the calculated area units.
