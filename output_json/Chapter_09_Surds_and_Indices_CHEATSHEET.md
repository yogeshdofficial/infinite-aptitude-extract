# Surds And Indices — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Base:** Scan for common bases (e.g., 2, 3, 5, 10) or cyclic variables.
2. **Standardize:** Convert all roots to fractional exponents ($\sqrt[n]{x} = x^{1/n}$) and decimals to fractions.
3. **Match Bases:** If $a^x = a^y$, set $x = y$. If bases differ, use logs or prime factorization to force a match.
4. **Apply Identities:** Use $(a \pm b)^2$ or $(a^n - b^n)$ patterns for complex surd expressions.
5. **Solve/Simplify:** Isolate the variable or simplify the expression using exponent laws.
6. **Sanity Check:** Verify signs, check for division by zero, and ensure the magnitude is logical.

## Formula Bank

**[Algebraic Simplification]**
$$x^{-n} = \frac{1}{x^n}$$
→ *produces: positive exponent form*
→ *use when: negative powers appear*

$$a^m \times a^n = a^{m+n}$$
→ *produces: combined base power*
→ *use when: multiplying same bases*

**[Exponent Laws]**
$$(a^m)^n = a^{mn}$$
→ *produces: single exponent*
→ *use when: power raised to power*

$$\sqrt[n]{a^m} = a^{\frac{m}{n}}$$
→ *produces: fractional exponent*
→ *use when: dealing with roots*

$$a^n \times b^n = (ab)^n$$
→ *produces: combined base*
→ *use when: exponents are identical*

$$(a-b)(a^2+ab+b^2) = a^3-b^3$$
→ *produces: difference of cubes*
→ *use when: simplifying complex binomials*

**[Solving Equations]**
$$a^x = a^y \Rightarrow x = y$$
→ *produces: linear equation*
→ *use when: bases are equal*

$$\left(\frac{a}{b}\right)^{-n} = \left(\frac{b}{a}\right)^n$$
→ *produces: inverted base*
→ *use when: negative exponent exists*

**[Surds and Roots]**
$$(a+b)^2 = a^2 + b^2 + 2ab$$
→ *produces: perfect square expansion*
→ *use when: finding square root of surd*

$$(a-b)(a+b) = a^2-b^2$$
→ *produces: difference of squares*
→ *use when: rationalizing denominators*

## Step Sequences

**Algebraic Simplification:** Factorize numerator → Cancel common terms → Simplify result.
**Exponent Laws:** Convert to prime bases → Apply power rules → Combine exponents → Solve.
**Solving Equations:** Equate bases → Set exponents equal → Solve linear equation → Verify.
**Surds and Roots:** Rationalize denominator → Apply algebraic identities → Simplify indices → Calculate.

## Fastest Tricks

* **Algebraic Simplification:** Cyclic variables $x=y^a, y=z^b, z=x^c \Rightarrow abc = 1$.
* **Algebraic Simplification:** $\frac{1-x}{x-1} = -1$.
* **Exponent Laws:** Group all terms by prime base before adding exponents.
* **Solving Equations:** $2^{x-1} + 2^{x+1} = 2^x(2^{-1} + 2^1) = 2^x(2.5)$.
* **Surds and Roots:** Compare $\sqrt[n]{a}$ by converting all indices to the LCM of $n$.

---

## Trap Watch

* **Algebraic Simplification:** $\frac{1-x}{x-1} = 1$ → use $-1$.
* **Exponent Laws:** $(a+b)^n \neq a^n + b^n$ → factorize instead.
* **Solving Equations:** $2^{x-1} + 2^{x+1} = 2^{2x}$ → factor out $2^x$.
* **Surds and Roots:** Forgetting to rationalize the denominator → multiply by conjugate.

---

## Last-Minute Reminders

Any non-zero number raised to the power of zero is one.
When bases are equal, exponents must be equal for the equation to hold.
Always convert decimals to fractions before applying exponent laws.
The order of a radical $\sqrt[n]{x}$ is $n$, which determines the fractional exponent.
Negative exponents indicate reciprocals, not negative values.
