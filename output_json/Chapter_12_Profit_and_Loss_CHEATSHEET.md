# Profit And Loss — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Actor:** Is it a simple transaction, a chain of sellers, or a dishonest dealer?
2. **Identify the Base:** Is the percentage based on Cost Price (CP) or Selling Price (SP)? (Default is CP).
3. **Normalize Units:** Ensure all quantities (kg, meters, items) are consistent before calculating price.
4. **Select Pattern:** Match the question structure to the Quick-Recognition Table.
5. **Set Equation:** Use the formula for the specific pattern; if multiple stages, use successive multipliers.
6. **Sanity Check:** If profit, SP > CP; if loss, SP < CP. Check decimal placement.

## Formula Bank

**[Basic Profit Loss]**
$$\text{Profit} = \text{S.P.} - \text{C.P.}$$
→ *produces: absolute profit value*
→ *use when: calculating raw gain*

$$\text{Profit } \% = \frac{\text{S.P.} - \text{C.P.}}{\text{C.P.}} \times 100$$
→ *produces: profit percentage*
→ *use when: comparing gain to cost*

**[Discount and Marked Price]**
$$\text{Net Price} = \text{M.P.} \times (1 - d_1) \times (1 - d_2)$$
→ *produces: final selling price*
→ *use when: applying successive discounts*

$$\text{Net Change} = x - y - \frac{xy}{100}$$
→ *produces: net percentage effect*
→ *use when: markup $x$ and discount $y$*

**[Dishonest Dealer]**
$$\text{Gain}\% = \frac{\text{Error}}{\text{True Value} - \text{Error}} \times 100$$
→ *produces: actual gain percentage*
→ *use when: using false weights*

**[Find CP from SP]**
$$\text{C.P.} = \frac{\text{S.P.}}{1 \pm \frac{r}{100}}$$
→ *produces: original cost price*
→ *use when: reversing profit/loss*

**[Mixture Profit Loss]**
$$\text{Mean Price} = \frac{\sum (\text{Quantity}_i \times \text{Rate}_i)}{\sum \text{Quantity}_i}$$
→ *produces: weighted average cost*
→ *use when: mixing different goods*

**[Multi-Stage Transactions]**
$$\text{Final Price} = \text{Initial} \times (1 \pm r_1) \times (1 \pm r_2)$$
→ *produces: final transaction price*
→ *use when: chain of sellers*

**[Price Change Scenarios]**
$$\frac{\text{S.P.}_1}{100 \pm r_1} = \frac{\text{S.P.}_2}{100 \pm r_2}$$
→ *produces: required new SP*
→ *use when: changing profit target*

**[Quantity-Price Equivalence]**
$$\text{Gain}\% = \frac{\text{Quantity Bought} - \text{Quantity Sold}}{\text{Quantity Sold}} \times 100$$
→ *produces: gain percentage*
→ *use when: SP of $x$ = CP of $y$*

## Step Sequences

* **Basic P/L:** Find difference → Divide by CP → Multiply by 100.
* **Discount/MP:** Apply successive discounts → Calculate final price.
* **Dishonest Dealer:** Calculate error → Divide by false weight → Multiply by 100.
* **Find CP from SP:** Identify P/L % → Divide SP by $(1 \pm r/100)$.
* **Miscellaneous:** Calculate total CP → Calculate total SP → Find difference.
* **Mixture P/L:** Find weighted average CP → Apply profit % → Get SP.
* **Multi-Stage:** Start with initial → Multiply by successive factors → Final.
* **Price Change:** Find CP → Apply new profit % → New SP.
* **Quantity-Price:** Equate total money → Find ratio → Calculate %.
* **Uncategorized:** Adjust quantity for loss/breakage → Calculate total CP/SP → Compare.

## Fastest Tricks

* **Equal P/L %:** Two items sold at same price with equal P/L % is always $\frac{x^2}{100}\%$ loss.
* **Markup/Discount:** Net profit = $\text{Markup} - \text{Discount} - \frac{\text{Markup} \times \text{Discount}}{100}$.
* **False Weight:** Gain % = $\frac{\text{Error}}{\text{False Weight}} \times 100$.
* **Quantity Equivalence:** If SP of $x$ = CP of $y$, Profit % = $\frac{y-x}{x} \times 100$.

---

## Trap Watch

* **Basic P/L:** Profit on SP → Use CP as denominator.
* **Discount:** Adding discounts → Apply successively.
* **Dishonest Dealer:** Error/1000 → Use Error/False Weight.
* **Multi-Stage:** Adding percentages → Multiply factors.
* **Uncategorized:** Broken items → Subtract from total quantity first.

---

## Last-Minute Reminders

* Always calculate profit and loss percentages on the Cost Price unless explicitly stated otherwise.
* When working with successive percentage changes, multiply the factors $(1 \pm r/100)$ rather than adding the rates.
* In dishonest dealer problems, the "Error" is the difference between true weight and false weight.
* For mixture problems, ensure the mean price is the Cost Price before applying the Alligation rule.
* If the question mentions overhead expenses, add them to the Cost Price before calculating profit.
