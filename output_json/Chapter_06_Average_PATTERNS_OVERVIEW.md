# Average — Chapter Overview

## What This Chapter Is About

This chapter deals with the concept of the "arithmetic mean," which represents a single value that summarizes a set of data by balancing out the differences between individual items. Every problem in this chapter revolves around the fundamental relationship between the total sum of a group and the number of items within that group. You will encounter questions ranging from finding simple averages of sets to solving for missing values, handling weighted data, and calculating changes when items are added, removed, or replaced.

---

## Core Concepts

**[The Balancing Act]** — An average is the "fair share" value. If you have a group of numbers, the average is the value each number would have if they were all made equal without changing their total sum. For example, if three people have 2, 4, and 9 apples, their total is 15; the average of 5 means that if they redistributed their apples equally, everyone would have 5.

**[The Sum-Average Link]** — The average is not a standalone number; it is a bridge between the total sum and the count of items. If you know any two of these (Average, Sum, Count), you can always find the third. For instance, if you know the average age of 10 students is 15, you immediately know their total age is $10 \times 15 = 150$.

**[Weighted Influence]** — Not all items in a set contribute equally to the average if they appear with different frequencies. When calculating the average of groups of different sizes, you must account for the "weight" (the count) of each group. For example, if a class of 20 students averages 60 marks and a class of 30 students averages 70, the overall average is closer to 70 because the second group is larger.

**[Property Transformation]** — Averages are sensitive to changes applied to the entire set. If you add, subtract, multiply, or divide every number in a set by a constant value, the average changes by that exact same operation. If you have an average of 10 and you add 5 to every number in the set, the new average will be exactly 15.

## Key Terms Glossary

**Observation** — Any single numerical value within the set being analyzed. In the set $\{10, 20, 30\}$, each number is an observation.

**Sum of Observations** — The result of adding all individual values in a set together.

**Number of Observations ($n$)** — The total count of items in the set.

**Consecutive Numbers** — A sequence of numbers where each number is exactly 1 greater than the previous one (e.g., 12, 13, 14).

**Weighted Average** — An average where some values contribute more to the final result than others due to their frequency or importance.

## Core Formulas

### Basic Average Formula

$$
\text{Average} = \frac{\text{Sum of observations}}{\text{Number of observations}}
$$

**Variables:**
- $\text{Sum}$ = Total value of all items
- $\text{Number}$ = Total count of items

**When to use:** When you need to find the central value of a static set of numbers.

**Key insight:** Always remember that $\text{Sum} = \text{Average} \times \text{Number}$; this rearrangement is used more often than the formula itself.

**Worked example:** Average of 10, 20, 30 is $\frac{10+20+30}{3} = 20$.

### Average Speed (Equal Distances)

$$
\text{Average Speed} = \frac{2xy}{x+y}
$$

**Variables:**
- $x$ = Speed for the first half of the distance
- $y$ = Speed for the second half of the distance

**When to use:** Only when the distance traveled at speed $x$ is exactly the same as the distance traveled at speed $y$.

**Key insight:** This is the harmonic mean of the two speeds; do not simply take the arithmetic average $\frac{x+y}{2}$.

**Worked example:** If you go to work at 30 kmph and return at 60 kmph, average speed is $\frac{2 \times 30 \times 60}{30+60} = 40$ kmph.

## Chapter-Wide Traps

**The "Average of Averages" Trap:** Assuming the average of two groups is the average of their individual averages → Always calculate the total sum of all items first, then divide by the total count.

**The "Arithmetic Mean" Speed Trap:** Using $\frac{x+y}{2}$ for average speed → Always use the harmonic mean formula $\frac{2xy}{x+y}$ when distances are equal.

**The "Missing Variable" Calculation Error:** Forgetting to multiply the average by the count to find the total sum → Always convert the average to a total sum immediately when an unknown value is involved.
