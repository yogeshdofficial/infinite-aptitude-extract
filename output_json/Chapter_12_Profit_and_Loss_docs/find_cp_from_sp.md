# Find CP from SP

## Overview
These questions provide the Selling Price (SP) and a percentage of profit or loss, asking you to reverse-calculate the original Cost Price (CP). The core idea is that CP is always the base ($100\%$), and SP is a scaled version of that base.

---

## Recognition Clues
* **Keywords:** "sold for", "paid", "sale was of", "incurred a loss", "charges... more than cost price".
* **Given:** SP value and a percentage (Profit% or Loss%).
* **Asked:** "What was the cost price?" or "At what price had she bought...".
* **Structure:** A single transaction where the relationship between SP and CP is defined by a percentage change.

---

## Important Formulas

### [Cost Price from Profit]
$$
\text{C.P.} = \text{S.P.} \times \left( \frac{100}{100 + \text{Profit}\%} \right)
$$
*Meaning:* Converts SP back to CP when a profit percentage is given.
*Use when:* The item is sold at a gain (e.g., "charges 23% more").

### [Cost Price from Loss]
$$
\text{C.P.} = \text{S.P.} \times \left( \frac{100}{100 - \text{Loss}\%} \right)
$$
*Meaning:* Converts SP back to CP when a loss percentage is given.
*Use when:* The item is sold at a loss (e.g., "incurred a loss of 15%").

---

## Solution Framework
1. Identify the SP and the percentage (Profit or Loss).
2. Determine the denominator: $(100 + \text{Profit}\%)$ or $(100 - \text{Loss}\%)$.
3. Set up the fraction: $\frac{100}{\text{Denominator}}$.
4. Multiply the fraction by the given SP.
5. Simplify the calculation to find the CP.

---

## Shortcut Tricks
* **Fractional Multipliers:** Convert the percentage to a fraction first (e.g., $20\%$ loss = $\frac{1}{5}$ loss, so $SP = \frac{4}{5}CP$).
* **Why:** It avoids large numbers and makes mental division easier.
* **Example:** For $20\%$ loss, $SP = \frac{4}{5}CP \Rightarrow CP = SP \times \frac{5}{4}$.

---

## Common Mistakes
* **Mistake:** Calculating the percentage of the SP instead of the CP.
    * *Reason:* Forgetting that profit/loss is always reckoned on CP.
    * *Fix:* Always write $CP \times (\text{multiplier}) = SP$ before solving.
* **Mistake:** Adding/subtracting the percentage directly from the SP (e.g., $SP - 20\% \text{ of } SP$).
    * *Reason:* Confusing the base of the percentage.
    * *Fix:* Use the formula $\frac{100}{100 \pm \%}$ to ensure the base is CP.

---

## Similar Patterns
This is distinct from "Profit/Loss % calculation" because here the CP is the unknown variable. If the question asks for "Profit %", you are finding a ratio; if it asks for "CP", you are reversing a percentage.

---

## Revision Summary
**Key formula:** $CP = SP \times \frac{100}{100 \pm \%}$.
**Spot it by:** Given SP and a percentage, asked for the original price.
**Fastest method:** Use fractional multipliers (e.g., $25\%$ profit = $\frac{5}{4}CP$).
**Biggest trap:** Calculating the percentage based on the SP instead of the CP.
