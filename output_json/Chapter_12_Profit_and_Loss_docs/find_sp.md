# Find SP

## Overview
This pattern involves calculating the final selling price when the cost price and a percentage gain or loss are provided. The central idea is to treat the cost price as the base ($100\%$) and adjust it by the given percentage to find the selling price.

---

## Recognition Clues
* **Keywords:** "Find S.P.", "What should be the selling price", "sold at a gain/loss of".
* **Given:** Cost Price (C.P.) and a percentage value (Gain % or Loss %).
* **Asked:** Selling Price (S.P.).
* **Scan for:** A fixed monetary value followed by a percentage change.

---

## Important Formulas

### Selling Price with Gain
$$
\text{S.P.} = \text{C.P.} \times \left( \frac{100 + \text{Gain}\%}{100} \right)
$$
*Meaning:* S.P. is the C.P. increased by the gain percentage.
*Use when:* The problem states a profit or gain percentage.

### Selling Price with Loss
$$
\text{S.P.} = \text{C.P.} \times \left( \frac{100 - \text{Loss}\%}{100} \right)
$$
*Meaning:* S.P. is the C.P. decreased by the loss percentage.
*Use when:* The problem states a loss percentage.

---

## Solution Framework
1. Identify the C.P. and the percentage (Gain or Loss).
2. If Gain, add the percentage to $100$; if Loss, subtract from $100$.
3. Convert this result into a decimal multiplier (e.g., $107\% = 1.07$).
4. Multiply the C.P. by this decimal to get the S.P.

---

## Shortcut Tricks
* **The Percentage Multiplier:** Instead of writing the fraction, multiply C.P. directly by $(1 \pm \text{decimal})$.
* **Why:** It eliminates the division step by $100$, allowing for faster mental calculation or single-line calculator input.

---

## Common Mistakes
* **Adding/Subtracting from C.P. instead of 100:** Adding the gain amount directly to C.P. (e.g., $56.25 + 20$).
    * *Reason:* Confusing absolute value with percentage.
    * *Fix:* Always convert the percentage to a multiplier first.
* **Miscalculating the decimal:** Using $1.7$ instead of $1.07$ for a $7\%$ gain.
    * *Reason:* Misplacing the decimal point in the percentage conversion.
    * *Fix:* Remember $100\% + 7\% = 107\%$, which is $1.07$.

---

## Similar Patterns
This is distinct from "Find CP" patterns, where you are given the S.P. and must divide by the percentage multiplier. Always check if you are moving from C.P. to S.P. (multiply) or S.P. to C.P. (divide).

---

## Revision Summary
**Key formula:** $\text{S.P.} = \text{C.P.} \times (1 \pm \text{rate})$.
**Spot it by:** Given C.P. and a percentage, asked for S.P.
**Fastest method:** Multiply C.P. by the percentage factor (e.g., $1.08$ for $8\%$ gain).
**Biggest trap:** Adding the percentage value directly to the C.P. instead of the base $100$.
