# Basic Profit Loss

## Overview
This pattern involves calculating profit/loss amounts or percentages when the Cost Price (CP) and Selling Price (SP) are provided. The central idea is that all profit or loss percentages are calculated relative to the original Cost Price, never the Selling Price.

---

## Recognition Clues
* **Keywords:** "purchased for," "sold for," "gain %," "loss %," "best transaction."
* **Given:** CP and SP (to find profit/loss %) OR CP and profit/loss % (to find SP).
* **Asked:** Profit/loss percentage, or the final Selling Price.
* **Comparison:** Questions asking to identify the "best transaction" among multiple options.

---

## Key Formulas

### Profit/Loss Percentage
$$
\text{Profit/Loss } \% = \left( \frac{\text{Profit or Loss}}{\text{C.P.}} \times 100 \right)
$$

**Variables:**
- $\text{Profit} = \text{S.P.} - \text{C.P.}$
- $\text{Loss} = \text{C.P.} - \text{S.P.}$
- $\text{C.P.}$ = Cost Price

**When to use:** When you have both CP and SP and need the percentage.
**Worked example:** CP = 12000, SP = 15000. Profit = 3000. Profit % = $(3000 / 12000) \times 100 = 25\%$.

### Selling Price from Percentage
$$
\text{S.P.} = \text{C.P.} \times \left( 1 \pm \frac{\text{Rate } \%}{100} \right)
$$

**Variables:**
- $\text{Rate } \%$ = The gain or loss percentage.
- Use $+$ for gain, $-$ for loss.

**When to use:** When you have CP and a gain/loss % and need the SP.
**Worked example:** CP = 56.25, Gain = 20%. SP = $56.25 \times (1 + 0.20) = 67.50$.

---

## Solution Framework

**Step 1: Identify CP and SP.** Clearly write down the given values to avoid confusion.
**Step 2: Calculate the difference.** Subtract the smaller value from the larger to find the absolute profit or loss.
**Step 3: Apply the percentage formula.** Divide the profit/loss by the **Cost Price** (not SP) and multiply by 100.
**Step 4: Adjust for multiple items.** If the question involves multiple items, calculate the total CP and total SP before finding the profit/loss.

---

## Shortcut Tricks

* **Trick:** To find SP quickly, multiply CP by $1.x$ for gain or $0.x$ for loss.
* **Why it works:** It combines the percentage calculation and addition/subtraction into one step.
* **When to use:** When calculating SP from CP and a percentage.
* **Example:** CP = 80.40, Loss = 15%. SP = $80.40 \times 0.85 = 68.34$.

---

## Common Mistakes

* **Mistake:** Using SP as the denominator for profit/loss %.
    * **Fix:** Always use CP as the base.
* **Mistake:** Adding/subtracting the percentage value directly to the currency (e.g., $100 + 20\% = 120$).
    * **Fix:** Calculate the percentage of the CP first, then add/subtract.
* **Mistake:** Using average prices for range-based questions.
    * **Fix:** Use the extreme values (min CP and max SP) to find the greatest possible profit.

---

## Worked Example (Step-by-Step)

**Question:** Mr. Kashyap purchased an air conditioner for ₹ 12000 and sold it for ₹ 15000. What was the profit percentage?

**Solution:**
1. **Identify:** CP = 12000, SP = 15000.
2. **Difference:** Profit = $15000 - 12000 = 3000$.
3. **Percentage:** Profit % = $(3000 / 12000) \times 100$.
4. **Simplify:** $(1/4) \times 100 = 25\%$.

**Answer:** 25%

## Revision Summary
* **Key formula:** $\text{Profit/Loss } \% = (\text{Difference} / \text{CP}) \times 100$.
* **Spot it by:** Presence of CP, SP, and percentage terms.
* **First move:** Calculate the absolute difference between CP and SP.
* **Fastest method:** Use multipliers (e.g., $1.20$ for $20\%$ gain).
* **Biggest trap:** Dividing by SP instead of CP.

## Appendix: Prerequisites

### Concepts You Must Know
* **Percentage** — A way of expressing a number as a fraction of 100.
* **Arithmetic** — Basic subtraction and division of large numbers.

### Formulas You Must Know First
* **Profit** = $\text{SP} - \text{CP}$
* **Loss** = $\text{CP} - \text{SP}$

### Terms Used In This Pattern
* **Cost Price (CP)** — The price at which an article is purchased.
* **Selling Price (SP)** — The price at which an article is sold.
