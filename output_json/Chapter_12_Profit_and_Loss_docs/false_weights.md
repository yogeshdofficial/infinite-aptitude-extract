# False Weights

## Overview
These questions involve a dealer who manipulates the weight scale or price to gain profit while claiming to sell at cost price. The central idea is to calculate the ratio of the "Actual Weight Used" to the "True Weight" and convert the resulting discrepancy into a percentage gain or loss.

---

## Recognition Clues
* **Keywords:** "Professes to sell at cost price," "uses a weight of X instead of Y," "false metre scale," "cheats while buying/selling."
* **Given:** The false weight used (e.g., 960g for 1kg) or the percentage of weight reduction.
* **Asked:** The actual gain/loss percentage or the specific length/weight used to achieve a target profit.

---

## Important Formulas

### [Basic Gain from False Weight]
$$
\text{Gain \%} = \left[ \frac{\text{Error}}{\text{True Value} - \text{Error}} \times 100 \right] \%
$$
*Meaning:* Error is the difference between True Value (e.g., 1000g) and False Weight used.
*Use when:* The dealer claims to sell at cost price but uses a lighter weight.

### [Combined Profit/Loss with False Weight]
$$
\text{Gain/Loss \%} = \left\{ \frac{y - x}{100 - y} \times 100 \right\} \%
$$
*Meaning:* $x$ is the % profit/loss on price; $y$ is the % reduction in weight.
*Use when:* The dealer changes both the price (profit/loss %) and the weight.

---

## Solution Framework
1. Define the "True Value" (usually 1000g or 100cm).
2. Identify the "False Value" actually used by the dealer.
3. Calculate the "Error" (True Value - False Value).
4. Treat the False Value as the effective Cost Price (C.P.).
5. Apply the gain formula: $\frac{\text{Profit}}{\text{C.P.}} \times 100$.

---

## Shortcut Tricks
* **The Ratio Method:** Always write the ratio as $\frac{\text{S.P.}}{\text{C.P.}}$ using the values: $\frac{\text{True Weight}}{\text{False Weight}} \times \frac{\text{Price Factor}}$.
* **Why it works:** It bypasses complex algebraic equations by treating the weight discrepancy as a direct multiplier to the profit margin.

---

## Common Mistakes
* **Denominator Error:** Using the "True Value" (1000) as the denominator instead of the "False Value" (e.g., 960).
    * *Reason:* Profit is calculated on the actual investment (what the dealer paid for), which is the weight he actually gave away.
    * *Fix:* Always use (True - Error) as the denominator.
* **Sign Confusion:** Mixing up the sign in the combined formula.
    * *Reason:* Forgetting that a "loss" on price is a negative value.
    * *Fix:* Always plug in $x$ as a negative number if it is a loss.

---

## Similar Patterns
* **Successive Percentage Changes:** Often confused with "cheating while buying AND selling."
* **Distinction:** If the question mentions "weights," use the False Weight formula; if it only mentions "price markups," use standard Successive Percentage formulas.

---

## Revision Summary
**Key formula:** $\text{Gain \%} = \frac{\text{Error}}{\text{False Weight}} \times 100$.
**Spot it by:** Phrases like "professes to sell at cost price" or "false weight."
**Fastest method:** Use the ratio $\frac{\text{True Weight}}{\text{False Weight}}$ and convert to percentage.
**Biggest trap:** Using 1000 as the denominator instead of the actual weight used.
