# Pattern Overview
This pattern provides a comprehensive framework for calculating financial outcomes—specifically profit, loss, and percentage variations—based on the relationship between the cost price and the selling price of goods. It is essential for mastering commercial mathematics, as it forms the foundation for understanding business transactions, retail pricing, and investment analysis.

# Key Formulas

**Basic Profit and Loss**
*   $$\text{Gain} = \text{S.P.} - \text{C.P.}$$
*   $$\text{Loss} = \text{C.P.} - \text{S.P.}$$

**Percentage Calculations**
*   $$\text{Gain \%} = \left( \frac{\text{Gain}}{\text{C.P.}} \times 100 \right)$$
*   $$\text{Loss \%} = \left( \frac{\text{Loss}}{\text{C.P.}} \times 100 \right)$$

**Finding S.P. or C.P. from Percentages**
*   $$\text{S.P.} = \text{C.P.} \times \left( \frac{100 + \text{Gain \%}}{100} \right)$$
*   $$\text{S.P.} = \text{C.P.} \times \left( \frac{100 - \text{Loss \%}}{100} \right)$$
*   $$\text{C.P.} = \text{S.P.} \times \left( \frac{100}{100 + \text{Gain \%}} \right)$$
*   $$\text{C.P.} = \text{S.P.} \times \left( \frac{100}{100 - \text{Loss \%}} \right)$$

**Special Cases**
*   **Common Loss (Two items at same x% gain/loss):** $$\text{Loss \%} = \left( \frac{x}{10} \right)^2$$
*   **False Weights (Gain %):** $$\text{Gain \%} = \left( \frac{\text{Error}}{\text{True Value} - \text{Error}} \right) \times 100$$

# When to Use This Pattern
*   **Keywords:** "Purchased," "Sold," "Profit," "Loss," "Gain percent," "Cost price," "Selling price," "False weight."
*   **Recognition:** Use this when a problem provides the price of acquisition and the price of disposal, or asks for the percentage change relative to the initial investment.
*   **Given/Found:** Typically, you are given two of the three variables (C.P., S.P., or Percentage) and asked to solve for the missing third.

# Core Concept & Theory
The fundamental principle of this pattern is that **all profit and loss percentages are calculated relative to the Cost Price (C.P.)**, not the Selling Price. By treating the C.P. as the "base" (100%), we can express the S.P. as a direct percentage of the C.P. This allows for rapid conversion between monetary values and percentage margins.

# Step-by-Step Solution Method
1.  **Identify Variables:** Clearly list the given C.P., S.P., or percentage values.
2.  **Determine the Goal:** Identify if you need to find the absolute profit/loss (currency) or the percentage.
3.  **Select Formula:** Choose the formula based on the known variables.
    *   If C.P. and S.P. are known, use the basic Gain/Loss formulas.
    *   If a percentage is involved, use the S.P./C.P. conversion formulas.
4.  **Execute Calculation:** Substitute values into the formula.
5.  **Verify:** Ensure the result makes logical sense (e.g., if S.P. > C.P., the result must be a gain).

# Worked Examples

**Example 1: Basic Profit Percentage**
*Given:* Mansi purchased a car for ₹2,50,000 and sold it for ₹3,48,000.
*Solution:*
1. C.P. = 2,50,000; S.P. = 3,48,000.
2. Gain = S.P. - C.P. = 3,48,000 - 2,50,000 = 98,000.
3. Gain % = (Gain / C.P.) × 100 = (98,000 / 2,50,000) × 100 = 39.2%.

**Example 2: Finding S.P. from C.P. and Gain %**
*Given:* An item costs ₹500 and is sold at a 20% profit.
*Solution:*
1. C.P. = 500; Gain % = 20%.
2. Use formula: S.P. = C.P. × ((100 + Gain %) / 100).
3. S.P. = 500 × (120 / 100) = 500 × 1.2 = ₹600.

# Recognition Clues & Keywords
*   **"Professes to sell at cost price":** Signals a false weight problem.
*   **"Two similar items":** Signals the common loss formula ($x^2/100$).
*   **"Reckoned on C.P.":** A reminder that the denominator is always the cost price.
*   **Avoid Confusion:** Do not confuse "Profit %" with "Profit amount." Always check if the question asks for a currency value or a percentage.

# Common Mistakes
1.  **Calculating % on S.P.:** Students often divide the profit by the Selling Price instead of the Cost Price. Always remember: Profit/Loss is relative to the investment (C.P.).
2.  **Ignoring the "False Weight" denominator:** In false weight problems, the denominator is (True Value - Error), not just the True Value.
3.  **Sign Errors:** In the formula for loss, ensure you subtract S.P. from C.P. to keep the value positive.
4.  **Misinterpreting "100 + Gain %":** Forgetting to add the percentage to 100 before multiplying by C.P.

# Practice Tips
*   **Memorize the 100% Base:** Think of C.P. as 100%. If you gain 20%, S.P. is 120%. If you lose 20%, S.P. is 80%. This mental shortcut is faster than using complex formulas.
*   **Related Patterns:** Study "Percentages" and "Ratio and Proportion" to improve your speed in manipulating these equations.
*   **Drill:** Practice converting fractions to percentages (e.g., 1/4 = 25%) to speed up calculations.
