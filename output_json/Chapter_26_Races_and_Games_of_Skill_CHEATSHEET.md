# Races And Games Of Skill — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Goal:** Does the question ask for a distance, a time, a speed, or a ratio?
2. **Check for "Start":** If a "start" is mentioned, identify who gives it and who receives it.
3. **Select Pattern:** Use the Quick-Recognition Table to map the question to one of the four patterns.
4. **Set up Ratios:** Convert all given data into a consistent ratio (Distance or Speed).
5. **Apply Formula:** Use the specific formula for that pattern to solve for the unknown variable.
6. **Sanity Check:** Ensure the winner covers the full distance and the loser covers less; check units (m vs km, sec vs min).

| Pattern | Trigger Words | Given | Find |

## Formula Bank

**[Chain Ratio Races]**
$$\frac{A}{C} = \frac{A}{B} \times \frac{B}{C}$$
→ *produces: Ratio of A to C*
→ *use when: Linking two separate race results*

$$\text{Start} = \text{Race Distance} - \text{Distance Covered}$$
→ *produces: The start distance*
→ *use when: Calculating handicap distance*

**[Percentage Fraction Games]**
$$\text{Percentage} = \frac{\text{Part}}{\text{Whole}} \times 100$$
→ *produces: Percentage of wins*
→ *use when: Calculating success rate*

$$\text{Simplified Fraction} = \frac{\text{Numerator} \div \text{GCD}}{\text{Denominator} \div \text{GCD}}$$
→ *produces: Reduced fraction*
→ *use when: Simplifying game ratios*

**[Race Time Distance]**
$$\text{Speed} = \frac{\text{Distance}}{\text{Time}}$$
→ *produces: Speed of contestant*
→ *use when: Time and distance are known*

$$\text{Distance} = \text{Speed}_B \times \text{Time Difference}$$
→ *produces: Distance of beat*
→ *use when: Given time difference for same distance*

**[Relative Speed Start]**
$$\text{km/hr} \times \frac{5}{18} = \text{m/sec}$$
→ *produces: Speed in m/s*
→ *use when: Converting units for race*

$$\frac{D_A}{S_A} = \frac{D_B}{S_B}$$
→ *produces: Time equality*
→ *use when: Both reach finish together*

## Step Sequences

**Chain Ratio:** Identify ratios → Multiply ratios → Scale to race distance → Find difference.
**Percentage Fraction:** Identify part/whole → Divide → Multiply by 100 → Simplify.
**Race Time Distance:** Find loser's speed → Calculate time difference → Multiply speed by time → Result.
**Relative Speed Start:** Convert units → Set time equality → Solve for distance/speed → Result.

---

## Fastest Tricks

**[Chain Ratio]:** Use $\frac{1000-x}{1000-y}$ for ratio of distances in a 1000m race.
**[Race Time Distance]:** Loser's speed = $\frac{\text{Distance of beat}}{\text{Time difference}}$.
**[Relative Speed Start]:** If $A:B$ speed ratio is $x:y$, then $A$ covers $x$ while $B$ covers $y$.

---

## Trap Watch

**[Chain Ratio]:** Adding distances instead of ratios → Use multiplicative ratios.
**[Percentage Fraction]:** Dividing total by part → Divide part by total.
**[Race Time Distance]:** Confusing A's time with B's → Label variables clearly.
**[Relative Speed Start]:** Adding start to A's distance → Subtract from B's distance.

---

## Last-Minute Reminders

* Always convert km/hr to m/s before calculating race times.
* A "start" of $X$ meters means the loser runs $(Total - X)$ meters.
* In a "beats by" question, the winner covers the full distance while the loser covers the remainder.
* If two people finish at the same time, their time ratio is $1:1$.
* Always verify if the race is 100m or 1000m (1km) before calculating.
