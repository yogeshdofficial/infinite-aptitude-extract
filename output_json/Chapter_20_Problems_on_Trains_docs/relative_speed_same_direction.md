# Relative Speed: Same Direction

## Overview
This pattern involves two objects (trains, people, or cyclists) moving in the same direction. The central idea is that their **relative speed** is the difference between their individual speeds, which effectively "slows down" the faster object relative to the slower one.

---

## Recognition Clues
* **Keywords:** "same direction," "overtakes," "passes," "running in the same direction."
* **Given:** Speeds of both objects (or one speed and the relative speed), lengths of objects (if trains).
* **Asked:** Time taken to cross, or the speed of one object.
* **Visual:** Imagine a train passing a man; the man is moving away, so the train has to "catch up" to him.

---

## Key Formulas

### Relative Speed (Same Direction)
$$
\text{Relative Speed} = \text{Speed}_{\text{faster}} - \text{Speed}_{\text{slower}}
$$

**Variables:**
- $\text{Speed}_{\text{faster}}$ = speed of the object doing the overtaking
- $\text{Speed}_{\text{slower}}$ = speed of the object being overtaken

**When to use:** Whenever two objects move in the same direction.

**Worked example:** Train at 68 kmph, man at 8 kmph. Relative speed = $68 - 8 = 60$ kmph.

### Time to Cross
$$
\text{Time} = \frac{\text{Total Distance}}{\text{Relative Speed}}
$$

**Variables:**
- $\text{Total Distance}$ = sum of lengths of both objects (if both are trains) or length of the train only (if passing a man).
- $\text{Relative Speed}$ = calculated using the formula above (ensure units are in m/sec).

**Worked example:** Train length 150m, relative speed 60 kmph ($60 \times \frac{5}{18} = \frac{50}{3}$ m/sec). Time = $150 \div \frac{50}{3} = 9$ seconds.

---

## Solution Framework
1. **Convert units:** Convert all speeds from km/hr to m/sec by multiplying by $\frac{5}{18}$.
2. **Calculate Relative Speed:** Subtract the slower speed from the faster speed.
3. **Determine Total Distance:** Use the length of the train(s) involved (if passing a man, distance = train length; if passing a train, distance = sum of both lengths).
4. **Apply Time Formula:** Divide the total distance by the relative speed to get the time in seconds.

---

## Shortcut Tricks
* **Trick:** Use the ratio of speeds if distance is constant.
* **Why it works:** Since $D = S \times T$, if $D$ is fixed, $S$ and $T$ are inversely proportional.
* **When to use:** When comparing two different scenarios (e.g., passing a platform vs. passing a man).
* **Example:** If a train passes a man in 12s and a platform in 20s, the ratio of times is $12:20$ or $3:5$.

---

## Common Mistakes
* **Mistake:** Adding speeds instead of subtracting.
    * **Why it happens:** Confusing "same direction" with "opposite direction."
    * **Fix:** Remember: Same direction = Subtract; Opposite direction = Add.
* **Mistake:** Forgetting to convert km/hr to m/sec.
    * **Why it happens:** Rushing the calculation.
    * **Fix:** Always check if the length is in meters and time in seconds.
* **Mistake:** Using only one train's length when passing another train.
    * **Why it happens:** Assuming only the moving train's length matters.
    * **Fix:** Always add the lengths of both objects being crossed.

---

## Worked Example (Step-by-Step)

**Question:** A 150-m long train is running at 68 kmph. In what time will it pass a man running at 8 kmph in the same direction?

**Solution:**
1. **Convert Speed:** $68 - 8 = 60$ kmph. $60 \times \frac{5}{18} = \frac{50}{3}$ m/sec.
2. **Distance:** Since it's a man, distance = train length = 150 m.
3. **Time:** $\text{Time} = \frac{150}{50/3} = 150 \times \frac{3}{50} = 9$ seconds.

**Answer:** 9 seconds.

---

## Similar Patterns
**Opposite Direction:** Distinguished by the keyword "towards each other" or "opposite direction"; use $\text{Relative Speed} = \text{Speed}_1 + \text{Speed}_2$.

---

## Revision Summary
* **Key formula:** $\text{Relative Speed} = S_1 - S_2$.
* **Spot it by:** "Same direction" keyword.
* **First move:** Convert km/hr to m/sec ($\times \frac{5}{18}$).
* **Fastest method:** Subtract speeds first, then divide distance by relative speed.
* **Biggest trap:** Adding speeds or forgetting to add lengths of two trains.

---

## Appendix: Prerequisites

### Concepts You Must Know
* **Speed-Distance-Time:** $\text{Distance} = \text{Speed} \times \text{Time}$.
* **Unit Conversion:** $1 \text{ km/hr} = \frac{5}{18} \text{ m/sec}$.

### Formulas You Must Know First
* $\text{Speed} = \frac{\text{Distance}}{\text{Time}}$
* $\text{Distance} = \text{Speed} \times \text{Time}$

### Terms Used In This Pattern
* **Relative Speed:** The speed of one object as observed from another moving object.
* **Overtake:** To catch up with and pass an object moving in the same direction.
