# Percentage — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Goal:** Is the question asking for a missing value, a percentage, or a change in a quantity?
2. **Map to Pattern:** Use the Quick-Recognition Table to match keywords to a specific pattern.
3. **Standardize Units:** Ensure all values (litres, kg, currency) are in the same units before calculating.
4. **Set Up Equation:** Use the identified formula to create an equation where the unknown is $x$.
5. **Execute Calculation:** Use shortcuts (fractions/multipliers) to solve for $x$.
6. **Sanity Check:** Does the answer make sense? (e.g., if a price increases, the new value must be higher than the original).

## Formula Bank

**[Basic Percentage Conversion]**
$$\text{Percentage} = \text{Fraction} \times 100\%$$
→ *produces: percentage value*
→ *use when: converting fractions to percent*

$$\text{Rate percent} = \text{Value} \times 100\%$$
→ *produces: rate percent*
→ *use when: converting decimals to percent*

**[Find Missing Value]**
$$\text{Total} = \frac{\text{Part}}{\text{Percentage}}$$
→ *produces: the whole value*
→ *use when: finding base from part*

$$\text{New Value} = \text{Original} \times (1 \pm \text{Rate})$$
→ *produces: adjusted value*
→ *use when: applying tax or increase*

**[Find Percentage]**
$$\text{Percentage} = \frac{\text{Part}}{\text{Whole}} \times 100\%$$
→ *produces: percentage share*
→ *use when: comparing two quantities*

$$\text{Percentage Error} = \frac{\text{Absolute Error}}{\text{Original Value}} \times 100$$
→ *produces: error percentage*
→ *use when: calculating measurement deviation*

**[Mixture and Solution]**
$$x = \frac{S(C_2 - C_1)}{100 - C_2}$$
→ *produces: amount to add*
→ *use when: increasing concentration*

$$\text{Pulp in Fresh} = \text{Pulp in Dry}$$
→ *produces: dry weight*
→ *use when: water evaporates*

**[Percentage Calculation]**
$$\text{Value} = \frac{\text{Percentage}}{100} \times \text{Total}$$
→ *produces: specific part*
→ *use when: calculating portion of total*

**[Set Theory]**
$$n(A \cup B) = n(A) + n(B) - n(A \cap B)$$
→ *produces: total unique items*
→ *use when: finding union of sets*

**[Successive Change]**
$$\text{Net change} = x + y + \frac{xy}{100}$$
→ *produces: total percentage change*
→ *use when: two successive changes occur*

$$\text{Required Increase} = \frac{x}{100-x} \times 100$$
→ *produces: restoration percentage*
→ *use when: reversing a reduction*

## Step Sequences

* **Basic Conversion:** Identify fraction → Multiply by 100 → Simplify.
* **Find Missing Value:** Set equation → Isolate $x$ → Solve.
* **Find Percentage:** Identify part/whole → Divide → Multiply by 100.
* **Mixture/Solution:** Identify constant component → Equate amounts → Solve for $x$.
* **Percentage Calc:** Convert % to fraction → Multiply → Add/Subtract.
* **Set Theory:** Sum individual sets → Subtract overlap → Subtract from total.
* **Successive Change:** Apply net change formula → Calculate final value.
* **Uncategorized:** Identify base → Apply rate → Calculate difference.

---

## Fastest Tricks

* **Basic Conversion:** Move decimal two places right for percent; two left for decimal.
* **Percentage Calc:** $16\frac{2}{3}\% = \frac{1}{6}$; $33\frac{1}{3}\% = \frac{1}{3}$; $12.5\% = \frac{1}{8}$.
* **Successive Change:** Use $x+y+\frac{xy}{100}$ for any two-step percentage change.
* **Mixture:** If water evaporates, equate the weight of the solid (pulp/salt).
* **Set Theory:** Total = (Sum of sets) - (Intersection).

---

## Trap Watch

* **Basic Conversion:** Confusing $0.2\%$ with $0.2$ → Divide by 100.
* **Find Missing Value:** Confusing part and whole → Identify base clearly.
* **Find Percentage:** Forgetting unit conversion → Convert all to same units.
* **Mixture:** Adding salt to numerator only → Add to denominator too.
* **Set Theory:** Forgetting to subtract overlap → Always subtract intersection.
* **Successive Change:** Adding percentages directly → Use $x+y+\frac{xy}{100}$.

---

## Last-Minute Reminders

* Always convert mixed fractions to improper fractions before calculating.
* Ensure the base (denominator) is the original value, not the new value.
* When water evaporates, the weight of the solid component remains constant.
* In set theory, the sum of individual sets always exceeds the total if there is an overlap.
* A percentage increase followed by an equal percentage decrease always results in a net loss.
