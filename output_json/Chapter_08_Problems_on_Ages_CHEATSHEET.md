# Problems On Ages — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify Timeframe:** Note if the question provides ratios/ages for the past, present, or future.
2. **Define Variables:** Assign $x$ to the smallest unit or the present age.
3. **Map Relationships:** Translate "times," "more than," or "less than" into algebraic equations.
4. **Select Pattern:** If ratios are given for two different time points, use **Ratio Evolution**. If specific age differences or products are given, use **Age Relationships**.
5. **Execute:** Solve the linear or quadratic equation for $x$.
6. **Verify:** Plug $x$ back into the required age expression and check if it satisfies all conditions.

## Formula Bank

**[Age Relationships]**

$$\text{Individual Age} = \frac{\text{Ratio Part}}{\text{Total Parts}} \times \text{Total Sum}$$
→ *produces: specific age of a person*
→ *use when: total sum and ratio are known*

$$\text{Age Difference} = \text{Age}_A - \text{Age}_B$$
→ *produces: constant difference value*
→ *use when: difference is fixed over time*

$$\text{Product of Ages} = A \times N = 240$$
→ *produces: age via quadratic equation*
→ *use when: product of two ages is given*

$$\text{Present Age} = \text{Age at Marriage} + \text{Years Married}$$
→ *produces: current age*
→ *use when: marriage age relation is given*

**[Ratio Evolution]**

$$\text{Age Ratio} = \frac{\text{Age}_1}{\text{Age}_2}$$
→ *produces: relative age proportion*
→ *use when: comparing two people's ages*

$$\text{Future Age} = \text{Present Age} + \text{Time Gap}$$
→ *produces: age at future point*
→ *use when: moving forward in time*

$$\text{Past Age} = \text{Present Age} - \text{Time Gap}$$
→ *produces: age at past point*
→ *use when: moving backward in time*

## Step Sequences

**Age Relationships:** Define variables → Set up equation → Solve for $x$ → Calculate target age.
**Ratio Evolution:** Define present ages as $ax, bx$ → Apply time shift → Equate ratios → Solve for $x$.

---

## Fastest Tricks

* **Ratio Evolution:** If ratio difference is constant, equate difference in parts to time gap.
* **Age Relationships:** Use the numerator of the fraction as the base unit for ratio simplification.
* **Age Relationships:** For $A \times N = \text{constant}$, test factors of the product that satisfy the linear condition.

---

## Trap Watch

* **Age Relationships:** Solving for man instead of mother → Read question target carefully.
* **Ratio Evolution:** Adding time to only one person → Add time to both ages.
* **Ratio Evolution:** Mixing up ratio order → Re-verify ratio sequence (e.g., Samir:Reema:Father).

---

## Last-Minute Reminders

The difference between two people's ages is always constant regardless of the time period.
When moving from "years ago" to "years hence," the total time gap is the sum of both values.
Always simplify ratios to their lowest terms before assigning $x$ units.
If a question mentions "product of ages," expect a quadratic equation and two possible roots.
Check if the question asks for the age "now" or "after/before X years."
