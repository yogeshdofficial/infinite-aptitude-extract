# Time And Distance — Exam Cheat Sheet

## How To Solve Any Question In This Chapter
1. **Identify the Goal:** Determine if you need Speed, Distance, or Time.
2. **Check Units:** Ensure all units are consistent (km/hr vs m/sec). Convert immediately.
3. **Identify Pattern:** Look for keywords (e.g., "average," "stoppage," "late/early," "relative").
4. **Select Formula:** Use the Formula Bank to pick the specific equation for that pattern.
5. **Set Up Equation:** If multiple variables exist, use ratios or algebraic equations ($D = S \times T$).
6. **Sanity Check:** Does the answer make physical sense? (e.g., speed shouldn't be negative).

## Formula Bank

**[Average Speed]**
$$\text{Average Speed} = \frac{\text{Total Distance}}{\text{Total Time}}$$
→ *produces: true average speed*
→ *use when: journey has multiple segments*

$$\text{Average Speed} = \frac{2v_1v_2}{v_1 + v_2}$$
→ *produces: average speed for two segments*
→ *use when: distance is equal both ways*

$$\text{Harmonic Mean} = \frac{n}{\sum \frac{1}{v_i}}$$
→ *produces: average speed for $n$ segments*
→ *use when: distances are equal*

**[Basic Speed Distance]**
$$\text{Speed (m/sec)} = \text{Speed (km/hr)} \times \frac{5}{18}$$
→ *produces: speed in m/sec*
→ *use when: units are mixed*

$$\text{Distance} = \text{Speed} \times \text{Time}$$
→ *produces: total distance covered*
→ *use when: speed and time known*

**[Complex Journeys]**
$$\text{Rate} = \frac{\text{Quantity}}{\text{Time}}$$
→ *produces: net flow rate*
→ *use when: dealing with leaks/pumps*

**[Relative Speed]**
$$\text{Relative Speed} = S_1 + S_2$$
→ *produces: closing speed*
→ *use when: moving in opposite directions*

$$\text{Relative Speed} = |S_1 - S_2|$$
→ *produces: closing speed*
→ *use when: moving in same direction*

**[Stoppage Problems]**
$$\text{Stoppage time} = \frac{V_1 - V_2}{V_1} \times 60$$
→ *produces: minutes stopped per hour*
→ *use when: comparing speeds with/without stops*

**[Time Variation]**
$$\text{Distance} = \frac{S_1 \times S_2}{|S_1 - S_2|} \times \Delta T$$
→ *produces: total distance*
→ *use when: late/early time given*

## Step Sequences

* **Average Speed:** Sum distances → Sum times → Divide total D by total T.
* **Basic Speed:** Convert units → Apply $D=ST$ → Solve for variable.
* **Complex Journeys:** Calculate net rate → Set up time equation → Solve for unknown.
* **Relative Speed:** Determine relative speed → Calculate distance gap → Divide gap by relative speed.
* **Stoppage:** Find speed difference → Divide by original speed → Multiply by 60.
* **Time Variation:** Calculate time difference → Apply distance formula → Solve for distance.

---

## Fastest Tricks

* **Average Speed:** Use LCM of speeds as total distance to find time fractions.
* **Relative Speed:** Ratio of speeds = Inverse ratio of time for same distance.
* **Time Variation:** If speed is $\frac{a}{b}$ of usual, time is $\frac{b}{a}$ of usual.
* **Stoppage:** Stops per hour = $\frac{\text{Speed Difference}}{\text{Original Speed}}$.

---

## Trap Watch

* **Average Speed:** Arithmetic mean of speeds → Use total D/T.
* **Relative Speed:** Forgetting head start distance → Subtract head start first.
* **Time Variation:** Mixing up late/early signs → Add if one late/one early.
* **Stoppage:** Including destination as a stop → Subtract 1 from total stops.

---

## Last-Minute Reminders

* Always convert minutes to hours by dividing by 60 before using speed formulas.
* Relative speed is additive for opposite directions and subtractive for same direction.
* The harmonic mean is always less than or equal to the arithmetic mean.
* When a train passes a pole, distance is train length; when passing a platform, distance is train + platform.
* If the question mentions "remaining distance," ensure you calculate the fraction of the total correctly.
